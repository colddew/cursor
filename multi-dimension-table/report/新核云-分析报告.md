# 新核云飞书客户案例分析报告

## 客户概述

**公司基本信息：**
- **公司名称**：上海纽酷信息科技有限公司（新核云）
- **行业**：工业软件/制造业数字化转型
- **规模**：全球服务3600+家制造业客户
- **主营业务**：面向离散制造业的新一代AI赋能的生产运营管理系统(MOM)，聚焦新能源汽车零部件、装备制造、3C智能硬件行业
- **核心价值**：利用工业大模型全面赋能制造业生产运营的软件研发商

## 飞书应用场景分析

### 1. 内部组织协同场景
- **产研项目管理**：将原本分散的N张表格聚合到飞书多维表格，实现项目全生命周期管理
- **文档管理**：基于飞书文档沉淀产品资料包，实现信息标准化和统一
- **团队协作**：从需求分析、开发排期到BUG收集的全流程协同

### 2. AI智能体应用场景
- **对外客户服务**：
  - 「质量合拍」Agent：处理质量异常通报
  - 「设备合拍」Agent：处理设备异常通报
- **对内效率提升**：
  - 「提个BUG」Agent：简化BUG提报流程
  - 「报工侠」Agent：优化报工流程
  - 「建个任务」Agent：快速任务创建
  - 「汇报助手」Agent：智能汇报总结

### 3. 生产协同场景
- **一生产单一群**：围绕生产单建立跨部门协作群
- **异常告警**：自动推送生产异常信息并@相关负责人
- **指标看板**：自定义生产指标看板并定期推送管理层

## 新获得的能力分析

### 1. AI驱动的智能化能力
**核心突破：**
- **自然语言处理能力**：AI智能体能够理解并处理自然语言消息，实现100%准确率的信息转化
- **实时响应能力**：机器人"秒回"客户消息，大幅提升响应速度
- **自动化数据录入**：将群聊中的非结构化信息自动转化为结构化表格数据

**具体表现：**
```
异常通报处理流程优化：
- 原流程：人工爬楼收集 → 手动录入文档 → 费力且不及时
- 现流程：AI自动识别 → 实时录入表格 → 100%准确率
```

### 2. 组织内外双向赋能能力
**对外赋能：**
- 为3600+制造业客户提供智能化异常处理方案
- 通过AI智能体帮助客户高效收集和分析生产各环节信息
- 实现客户生产问题的快速定位和解决

**对内赋能：**
- 内部团队协作效率提升40%
- 制造用户生产协作效率提升25%
- 实现4套业务软件ALL In One整合

### 3. 数据驱动决策能力
**管理洞察提升：**
- 生产数据实时感知，减少"救火"式管理
- 自定义指标看板，管理层定期获取生产整体情况
- 跨部门信息透明化，生产、质量、采购、设备等部门实时协同

### 4. 业务流程重构能力
**流程优化成果：**
- 产研项目进度管理从分散到集中，一目了然
- 产品信息资产统一沉淀和传播
- 异常处理从被动响应到主动预防

## 未满足的需求分析

### 1. 系统集成深度需求
**当前状态：**
- 新核云MES系统与飞书仍为两个独立系统
- 需要通过API打通实现数据交互
- 存在"多系统交互不统一"的挑战

**未满足需求：**
- 更深度的系统原生集成
- 无缝的用户体验过渡
- 统一的身份认证和权限管理

### 2. 行业特定功能需求
**制造业特殊需求：**
- 更复杂的生产排程优化算法
- 设备预测性维护能力
- 供应链协同的深度集成
- 质量追溯的全流程可视化

### 3. AI能力扩展需求
**智能化深化需求：**
- 更复杂生产异常的AI诊断能力
- 基于历史数据的生产优化建议
- 跨工厂、跨区域的智能调度
- 供应链风险的AI预警系统

### 4. 移动端和现场作业支持
**现场作业痛点：**
- 生产车间网络环境限制
- 移动设备操作的便利性需求
- 离线模式下的数据同步问题
- 现场人员的技术接受度培训

## 战略价值与借鉴意义

### 1. AI智能体企业应用的标杆意义
**创新模式：**
- **对话式交互**：将复杂的业务操作转化为自然语言对话
- **场景化AI**：针对具体业务场景开发专用智能体
- **双向赋能**：同时对内提升效率、对外服务客户

**借鉴价值：**
- 其他企业可参考这种"小而专"的AI智能体开发模式
- 证明了AI在企业级应用中的实用价值
- 为传统行业的AI化改造提供了可行路径

### 2. 制造业数字化转型的启示
**关键洞察：**

- **数据实时化**：生产数据从定期查看到实时感知的转变
- **决策智能化**：基于AI的异常处理和决策支持

**行业影响：**
- 为制造业企业提供了数字化转型的新思路
- 证明了协同平台与专业系统的结合价值
- 展示了AI在传统制造业中的落地应用

### 3. 产品生态协同的商业价值
**生态协同模式：**
- **产品互补**：新核云MES + 飞书协同平台 = 完整解决方案
- **客户价值倍增**：1+1>2的客户价值创造
- **市场竞争力提升**：通过生态合作增强市场地位

**商业启示：**

## 结论与建议

### 核心结论
新核云案例展示了一个成功的AI赋能数字化转型模式，其核心价值在于：
1. **AI智能体的创新应用**：通过场景化AI智能体实现效率倍增
2. **组织内外双向赋能**：同时提升内部效率和客户服务能力
3. **生态协同的商业模式**：通过与飞书的深度合作创造更大价值

2. **重视用户体验**：确保AI应用真正简化而非复杂化用户操作
3. **构建数据基础**：为AI应用建立高质量的数据基础
4. **培养AI思维**：在组织内部培养AI应用的思维和能力

### 未来发展方向
1. **AI能力深化**：从简单自动化向智能决策演进
2. **生态扩展**：构建更完整的制造业数字化生态
3. **标准化复制**：将成功经验标准化，向更多客户推广
4. **国际化布局**：借助数字化转型经验开拓国际市场
