# 速腾聚创飞书客户案例分析报告

## 📊 客户基本信息

**公司名称**：速腾聚创（RoboSense）
**所属行业**：智能出行/机器人技术
**公司规模**：501-2000人
**主营业务**：AI驱动的机器人技术公司，专注于激光雷达研发制造
**公司背景**：
- 成立于2014年，香港上市公司（2498.HK）
- 2025年摩根士丹利全球人形机器人百强榜企业
- 2024年/2023年激光雷达总销量第一
- 2022年专精特新"小巨人"企业
- 全球合作：290+汽车整车厂及Tier1，2600+机器人客户
- 业务覆盖：全球40多个国家和地区

## 🚀 飞书给客户的新能力/赋能

### 1. 智能工厂运营能力（核心价值）

#### "人机料法环"透明管理系统
**工厂运营提效60%**：从60分钟缩短至20分钟的数据统计时间

**"人" - 人员配置与追踪**
- 通过多维表格打造工时管理系统与人力看板
- 实时关联人员工作状态和任务进度
- 自动追踪任务耗时及工作饱和度
- 异常工时预警助管理者发现分配失衡
- 可视化展现成员进度与投入产出比

**"机" - 维保监控改善闭环**
- 设备OEE数据统计表格：精细统计生产车间设备效率信息
- 设备耗材监控表格：建立详尽的设备耗材台账
- 设备组提案改善汇总表：通过自动化功能按时提醒推进计划
- 实现设备维保、效率监控及改善措施推进的完整闭环管理

**"料" - 材料检验优化**
- IQC阶段标准化异常反馈收集
- 按模板填写并推送至指定飞书群及@相关人员
- 多维表格生成异常清单作为经验教训库
- 监控供应商来料质量，防止问题再发
- 为新项目风险评估提供历史数据

**"法" - 生产计划与调度优化**
- 构建"产能数据汇总表"：整合日计划达成、生产报表、产线节拍等重要信息
- 仪表盘技术直观呈现运营数据
- 自动化手段提高信息推送效率与精准度
- 数据更新后迅速通知相关群组和人员

**"环" - 现场5S管理**
- 使用多维表格数据库汇总任务及项目信息
- 记录5S问题点的物理属性分类、异常类型分类等信息
- 仪表盘实现5S问题不同维度的数据分析与可视化展示
- 自动化功能周期性提醒对应责任人

### 2. 平台化管理体系

**项目管理平台**
- 使用飞书多维表格实现复杂业务计算逻辑
- 提高项目管理、任务追踪的效率和精度
- 提高业务数据和信息的统一性

**数据监控平台**
- 仪表盘将运营数据与分析结果可视化
- 多个多维表格数据库间的联动和交互
- 数据指标更加凝练，而非简单的基础数据展示

**流程管理平台**
- 设置十多项自动化流程的定期推送
- 将工厂运营不同模块数据按天、按周推送至相关人员
- 提高工厂端信息对齐拉通的效率

### 3. 采购交付异常处理能力

**提效60%的采购异常处理**
- 统一异常反馈模版，自动补充关联信息
- 8个自动化场景，精准通知责任人处理异常
- 一键催办和转办功能
- 构建库存管理仪表盘，清晰快速反映库存结构和趋势

### 4. LTC全流程管理能力

**线索管理**
- 线索统一管理，避免多人同时跟进
- 自动提醒，降低沟通成本
- 仪表盘实时查阅线索跟进情况、闭环情况、转化率
- **节省时间：12.83小时/月**

**逾期应收款管理**
- 多维表格功能自动计算生成应收、逾期清单
- 配置对账单格式并用飞书邮件发给客户
- **节省时间：161小时/月**

**考核管理**
- 多维表格功能自动生成核算表
- 配置仪表盘视图，减少excel数据处理时间
- **节省时间：41.84小时/月**

### 1. 专业制造业深度集成需求

**生产执行系统(MES)集成**
- 案例中未提及与MES系统的深度集成
- 仍需手动更新部分生产数据到飞书系统
- 设备OEE数据需要人员实时更新，说明缺乏自动采集能力

**专业设备连接能力**
- 设备数据需要人工录入而非自动采集
- 缺乏与工业IoT设备的直接连接能力
- 生产车间实时数据获取仍存在延迟

### 2. 高级数据分析能力

**预测性分析功能**
- 缺乏设备故障预测功能
- 未提及生产质量趋势预测
- 缺少基于历史数据的智能决策支持

**高级商业智能**
- 仪表盘主要展示基础指标和过程数据
- 缺乏深度的业务洞察和智能分析
- 未提及AI驱动的智能推荐功能

### 3. 行业专业化功能

**激光雷达行业特定需求**
- 未提及针对激光雷达研发的专业化管理功能
- 缺乏针对高科技制造业的研发项目管理优化
- 质量管理体系与飞书的集成度有限

**供应链深度管理**
- 主要聚焦内部工厂管理，对上下游供应链的协同有限
- 缺乏与供应商系统的深度集成
- 全球化供应链管理能力有待加强

### 4. 移动端专业化

**生产环境专用应用**
- 未提及专门为生产车间设计的移动应用
- 缺乏工业级移动设备适配
- 生产现场移动操作能力有限
