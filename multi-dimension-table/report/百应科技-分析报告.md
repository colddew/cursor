# 百应科技-分析报告

## 客户基本信息

**公司名称**: 百应科技
**所属行业**: 企业服务（对话式AI应用领域）
**公司规模**: 成长型组织，连续六年保持增长，复合增长率（CAGR）超100%
**主营业务**: 对话式AI应用领域创领者，致力于将人工智能和大数据技术与政企多业务场景深度融合，助力政府沟通服务、企业增长的数智化升级。为客户提供全栈式AI数字员工解决方案，聚焦零售、金融和政务三类场景。
**企业背景**: 连续四年被认证为杭州准独角兽企业，拥有中美双AI研究院，百余项AI及大数据领域相关专利和软件著作。

## 飞书给客户的新能力/赋能

### 1. 数据管理效率革命
- **"数据驾驶舱"建设**: 使用飞书多维表格搭建统一数据管理平台，信息流转效率提升96.8%
- **一站式数据输入**: 一线员工可以在单一入口完成数据输入，缩减系统切换所耗费的时间，优化数据填报体感
- **实时数据可视化**: 数据"输入即输出"，无需进行二次加工处理，即能实现可视化呈现
- **全局洞察管理**: 管理者只需打开一个多维表格界面，就能在多个实时更新的数据看板中自由切换，把握和洞察全局更加丝滑和高效

### 2. 招聘流程全自动化
- **招聘全流程打通**: 使用飞书招聘打通招聘流程各个环节，招聘效率提升30%
- **智能日程管理**: 面试日程能够清晰地被展示在日历上，通过消息流及时提醒面试官
- **自动化审批**: offer审批实现自动提醒，减少人工干预
- **信息一键流转**: SSC登录飞书人事即可看到新同学的基本信息，省去了二次填报的时间
- **智能标签识别**: 通过自动提炼应聘者的学校、公司等标签，让了解候选人变得更轻松
- **标准化工具支持**: 飞书内嵌的模版、题库也能覆盖使用者的大多数需求

### 3. 企业文化数字化建设
- **"百应圈"社交平台**: 使用飞书公司圈功能打造"百应圈"，建设"公司的朋友圈"
- **自由表达空间**: 为员工提供表达的自由空间，促进自下而上的文化建设
- **公益文化凝聚**: 通过公益等生活化内容凝聚团队，如为流浪猫起名、建档立卡、修缮新窝等
- **文化传承新模式**: "将理念沉淀到工具"，再"用工具凝聚组织"的新模式
- **感恩文化萃取**: 感恩文化在"百应圈"的萃取，为公司员工从事公益活动搭建平台

## 飞书没有满足客户需求的部分

### 1. 初始系统架构分散问题
- **系统孤岛挑战**: 使用飞书前，不同部门的系统彼此独立，信息分散在各处
- **数据同步困难**: 财务、销售、客户成功部门都有各自的系统，填报、同步与汇总费时费力
- **数据质量隐患**: 系统彼此独立，管理者能够查看的数据就是点状和分散的，也不能实时更新
- **决策效率低下**: 数据同步和汇总时，需要花更多时间在流转和核对上

### 2. 招聘与办公系统割裂
- **系统互不连通**: 招聘系统与办公软件不互通，形成信息孤岛
- **人工操作繁琐**: 面试信息不能被自动展现在日历上，面试官容易忘记已预订的面试
- **重复工作问题**: 招聘与入职之间未搭建信息一键流转的通路，需要手动导出新人信息，打包发送给SSC
- **入职流程冗长**: 新同学入职时，SSC再让其填一次表，拉长了信息流转的时间

### 3. 企业文化传承挑战
- **规模扩张难题**: 随着公司体量扩张、人数急剧增长，团队原有的文化逐渐被"稀释"
- **传统模式局限**: "自上而下"的公司文化不能满足凝聚团队的需求
- **新理念缺乏载体**: 新的理念和共识则慢慢地、自下而上地生长，缺乏有效的凝聚工具
- **团队凝聚力下降**: 团队规模扩大后，原有的文化建设方式无法适应新的组织形态
