# 华诺智胜飞书客户案例深度分析报告

## 1. 公司基本信息

### 1.1 公司概况
- **公司名称**: 华诺智胜
- **成立时间**: 2017年
- **行业领域**: 医疗健康/互联网医疗
- **主营业务**:
  - 代谢性疾病创新管理平台（糖尿病、超重肥胖合并PCOS等代谢疾病）
  - 为医疗机构提供「数字慢病管理软件平台」
  - 为代谢病患者提供具备治疗效果的「生活方式数字疗法」
- **业务模式**: B2B2C运营模式
- **员工分布**: 成都、重庆、北京、天津
- **公司定位**: 国内首个糖尿病、体重数字化管理平台

### 1.2 业务特点
- 业务链条长，涉及多学科专业知识
- 面向C端用户提供全周期管理服务
- 需要跨部门协作与多学科知识汇总
- 创业公司，资源有限但业务复杂度高

## 2. 飞书应用场景

### 2.1 核心应用场景
华诺智胜将飞书应用于三个主要场景：

#### 场景一：组织文化建设和工作方式转型
- **"飞阅会"**: 采用文档默读形式提升开会效率
- **云文档与即时沟通**: 保障团队协作的高效顺畅
- **「华诺发布」订阅号**: 每日早上八点推送华诺金句，帮助员工快速进入工作状态
- **目标**: 打造适合数字时代的工作方式和组织文化

#### 场景二：患者全周期管理系统
- **飞书多维表格**: 搭建患者全周期管理系统
- **临床业务部**: 通过多维表格问卷形式为新患者建档
- **自动提醒机制**: 提醒健辅中心顾问进行患者跟进
- **信息同步**: 患者档案同步至数据表，定期维护更新
- **多维仪表盘看板**: 清晰展现所有信息，辅助员工把握患者全方位健康管理

#### 场景三：知识管理与员工培训
- **新手村知识库**: 沉淀企业内部知识
- **飞书词典词条**: 创建医疗专业知识相关词条
- **自主学习**: 新员工能快速自主学习公司文化理念、业务模式、经验方法等
- **培训质量监控**: 讲师通过学员文档笔记掌握真实培训成效

### 2.2 具体功能应用
- 飞书视频会议
- 飞书文档
- 即时信息
- 飞书多维表格
- 飞书知识库

## 3. 新获得的能力

### 3.1 数字化协作能力
- **跨部门协作效率提升**: 80%+的跨部门沟通效率提升
- **患者管理数字化**: 近万名糖尿病患者通过多维表格进行全周期管理
- **工作专注度提升**: 通过每日金句推送和工作生活区分，提升员工专注度
- **信息可追溯性**: 解决了原来信息传递繁复杂乱、难以追溯来源的问题

### 3.2 知识管理能力
- **知识体系化沉淀**: 企业内部知识和经验方法论得到有效沉淀
- **快速学习能力**: 新员工能快速自主学习相关知识
- **培训周期缩短**: 新员工培训时间从原先超过1个月以上缩减至2周
- **培训质量提升**: 讲师能够通过学员的文档笔记掌握真实培训成效

### 3.3 业务流程优化能力
- **患者管理流程优化**: 单位患者跨部门协作沟通时间从原来的每周3小时以上大幅缩减至30分钟
- **自动化工作流**: 自动提醒、自动同步等功能减少人工操作
- **数据可视化**: 多维仪表盘看板功能清晰展现所有信息
- **隐私保护**: 在飞书平台上更好地保障患者隐私

### 3.4 组织文化建设能力
- **工作方式现代化**: 向飞书对于工作细节的极致追求靠拢学习
- **组织凝聚力提升**: 通过统一的协作平台增强团队协作
- **工作氛围营造**: 通过每日金句推送营造浓厚的工作氛围
- **人才培养体系**: 建立了系统化的员工培训和能力发展体系

## 4. 未满足的需求

- **人才即插即用困难**: 新员工上任需要耗费大量时间从头学起
- **跨部门协作复杂性**: 糖尿病患者全周期管理的跨部门协作信息传递繁复杂乱

- **数据安全**: 患者隐私数据的安全保护需要更高级别的安全保障
- **系统集成**: 可能需要与现有医疗系统或专业软件的集成

### 4.3 业务扩展需求
