# 优赛诺飞书客户案例分析报告

## 1. 公司基本信息

### 公司概况
- **公司名称**: 优赛诺 (Ucello)
- **成立时间**: 2020年12月
- **所属行业**: 生物制药/医疗健康
- **公司规模**: 初创企业（具体规模未明确提及，但从案例描述可推断属于中小型生物技术企业）
- **主营业务**: 异体通用型CAR-T细胞治疗药物的研发及产业化

### 核心产品与技术
- 依托自主研发技术平台
- 开发出全球首个基于脐血来源T细胞的通用型CAR-T细胞治疗产品UC101
- 在免疫细胞治疗领域持续开拓创新
- 推进重大新药创制，搭建"产学研医用"运营模式

### 企业愿景
- 致力于成为新一代通用型CAR-T技术的创领者
- 建立"有追求、有效率、有温度"的企业文化
- 目标在未来2-3年内实现组织管理的全面优化

## 2. 飞书应用场景

### 2.1 组织管理场景
- **目标对齐**: 使用飞书OKR实现CEO关键目标到各部门具体行动的逐层分解
- **绩效管理**: 通过飞书绩效解决药企研发团队绩效评价困难的问题
- **组织文化建设**: 利用定制工作台和飞书直播传递企业文化

### 2.2 法务协同场景
- **合同审批**: 使用飞书合同打通企业内部合同审批系统
- **多方协商**: 支持相关部门、第三方和法律顾问同时参与合同协商修订
- **版本管理**: 解决合同文本反复传递修改导致的版本混乱问题

### 2.3 内部沟通场景
- **信息发布**: 通过定制工作台实现企业动态、新闻资讯、培训计划等多维度信息展示
- **直播互动**: 利用飞书直播进行《读书分享会》、《中秋Party》等企业活动
- **应急沟通**: 在疫情期间通过直播传递医疗知识、公司政策和员工经验

### 2.4 人才管理场景
- **招聘管理**: 使用飞书招聘进行人才招聘
- **员工培训**: 通过工作台培训计划板块进行员工培训
- **知识管理**: 利用飞书文档进行知识沉淀和分享

## 3. 新获得的能力

### 3.1 战略目标管理能力
- **目标对齐机制**: 实现了从CEO愿景到员工具体行动的有效转化
- **OKR式KPI体系**: 为基层员工量身打造适合研发特点的绩效评价体系
- **闭环绩效管理**: 建立全员参与的绩效管理评价流程，提升企业管理成熟度

### 3.2 法务流程数字化能力
- **多方协同审核**: 实现合同相关部门、第三方、法务的同步参与
- **审计合规保障**: 通过版本对比功能避免合同盖章版本与审核版本不一致的风险
- **流程效率提升**: 显著降低沟通时间成本，提高合同修订和审核效率

### 3.3 企业文化传播能力
- **多端适配展示**: 通过定制工作台实现信息的多端同步展示
- **实时互动沟通**: 利用飞书直播实现企业与员工的实时互动
- **情感连接建立**: 在特殊时期（如疫情）通过数字化手段传递企业关怀

### 3.4 组织敏捷性能力
- **快速信息传递**: 实现企业要闻、政策、活动的及时传达
- **灵活工作模式**: 支持远程办公场景下的组织协同
- **学习型组织建设**: 通过直播分享等活动促进知识分享和学习氛围

## 4. 未满足的需求

### 4.1 绩效评价客观性不足
- **行业特性挑战**: 案例中明确提到"药企研发团队很难客观评价绩效"是行业内最具共识的管理挑战
- **解决方案局限性**: 虽然飞书绩效提供了基础框架，但对于研发过程与结果的高度不确定性，评价体系仍有提升空间

### 4.2 管理经验缺乏
- **团队能力短板**: 新组建的管理团队主要具备科研背景，但在企业管理方面缺乏经验
- **数字化工具依赖: 严重依赖飞书等数字化工具来弥补管理经验的不足**

### 4.3 信息触达效果有限
- **传统方式失效**: 此前尝试的内部论坛网站浏览量有限，缺乏适当的激励引导
- **信息过载风险**: 员工面临信息洪流，可能导致重要信息被忽视

### 4.4 应急场景适应性
- **突发情况应对**: 在疫情管控和高温限电等特殊情况下，数字化工具的应急使用仍需优化
- **员工情感连接**: 远程办公加剧了员工的孤独感和隔阂情绪，数字化手段对情感连接的支持有限

### 4.5 行业特定需求
- **专业系统整合**: 生物制药企业的专业研发管理系统与飞书的整合需求未明确提及
- **合规性要求**: 医药行业的特殊合规要求在通用型办公平台中的适配性有待提升
