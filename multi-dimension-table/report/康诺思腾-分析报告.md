# 康诺思腾飞书客户案例深度分析报告

## 1. 公司基本信息

### 1.1 企业概况
- **公司名称**: 康诺思腾 (Cornerstone Robotics)
- **成立时间**: 2019年9月
- **行业**: 医疗健康 / 医疗器械 / 手术机器人
- **企业定位**: 中国创新型手术机器人行业领导者
- **规模**: 全球设立四大研发中心，七大业务中心

### 1.2 主营业务
- 专注于打造安全高效的手术机器人平台
- 自主研发腔镜手术机器人
- 提升高端医疗资源的可及性

### 1.3 企业荣誉
- 2024年胡润全球瞪羚企业
- 粤港澳大湾区生物科技创新50强"领军企业"
- 腔镜手术机器人于2024年9月获得NMPA批准上市

## 2. 飞书应用场景

### 2.1 核心应用领域
康诺思腾围绕三大核心场景构建管理体系：
- **研发管理**: 项目管理、风险控制、流程标准化
- **人力资源**: 全生命周期人事管理
- **流程控制**: 变更管控、质量保证

### 2.2 具体应用场景

#### 2.2.1 研发项目管理
- **周报自动化生成**: 整合20多个数据源，从被动汇总变为主动洞察
- **项目智能分析**: 建立标准化健康评估体系，实现"可对比、可诊断、可决策"
- **AI风险识别**: 自动分析项目群聊中的高频问题和潜在风险
- **角色化信息推送**: 高管看态势，执行层看重点

#### 2.2.2 变更管控系统
- **九类变更类型**: 智能匹配SOP流程
- **影响评估矩阵**: 自动派生测试任务与子工单
- **全链路流程**: 从需求提出到发布交付的完整管控
- **统一入口管理**: 数据结构化沉淀，角色权限控制

#### 2.2.3 人事一体化管理
- **全生命周期管理**: 从招聘到离职的完整流程
- **多模块整合**: 招聘、入职、假勤、薪酬、绩效自动衔接
- **数据驱动决策**: 全员数字档案、智能分析看板、自动化预警
- **流程效率提升**: Offer到离职的全流程贯通与数据同步
