# 永卓控股飞书客户案例分析报告

## 客户基本信息

**公司名称**：永卓控股（前身永钢集团）

**所属行业**：制造业（钢铁、新能源、新材料、金融贸易等多元产业）

**公司规模**：

- 13个二级单位
- 20余家工厂
- 26位生产计划专员
- 5个职能部门、9个职能科室、4大生产厂、13个生产分厂/车间

**主营业务**：钢铁、新能源、新材料、金融贸易等多元产业协同发展

**成立时间**：1984年（村办企业）

**公司地址**：江苏张家港市永联村

## 飞书给客户带来的新能力和价值

### 具体使用的飞书功能模块

1. **飞书多维表格**
   - 核心功能，被广泛应用于各个业务场景
   - 在13个二级单位全面使用
   - 搭建了数百个多维表格业务系统

2. **飞书OKR**
   - 用于冬储任务管理和目标对齐
   - 形成OKR管理树，层层分解目标

3. **飞书审批**
   - 作业安全分析审批流程
   - 自动化审批流程管理

4. **飞书文档**
   - 标准化生产知识库
   - 文档协作和信息共享

5. **飞书知识库**
   - 沉淀业务经验和知识
   - 常见作业步骤、危害和预防控制措施知识沉淀

6. **飞书会议室**
   - 支持协作会议和沟通

7. **飞书绩效**
   - 支持绩效管理流程

8. **即时信息/时间流提醒**
   - 自动提醒和任务通知

9. **飞书服务台**
   - 支持服务请求和处理

### 解决的业务痛点

1. **项目管理痛点**
   - 进度管理耗时耗力
   - 成本管理缺乏跟踪监测
   - 质量管理参差不齐
   - 成果管理冗杂无序

2. **生产计划编排痛点**
   - 完全串行操作，靠人力沟通
   - 依赖经验纠偏
   - 计划值与实际值符合率相差大

3. **冬储管理痛点**
   - 策划周期长
   - 员工自行梳理内容易缺失
   - 任务跟踪无法有效监管
   - 沟通成本高

4. **作业安全分析痛点**
   - 纸质表单填写困难
   - 表单整理繁琐
   - 人工提醒流程缓慢

### 带来的业务价值和效率提升

1. **项目管理效率提升**
   - 降低管理难度：基于可视化看板和预警提醒功能，实现"跨要素、跨界面、跨空间"融合协同
   - 减少沟通成本：将串联工序转变为并联工序，多处精益改善同时开展
   - 节约成本：零成本开发原本需要15万元采购的项目管理工具
   - 管理工时节省：平均单个项目节约3人/天管理工时、1.75人/天一线信息整理工时
   - 管理半径扩大：从20多项扩大至66项
   - 目标效益：项目管理目标效益高达5.4亿元

2. **生产计划编排效率提升**
   - 信息集成：生产计划平衡看板"all in one"
   - 自动化：自动平衡一稿过，减少人工干预
   - 时间节省：每天节省48小时，相当于减少2个人工
   - 经济效益：年价值达到30万元

3. **冬储管理效率提升**
   - 全局协调：形成完整的OKR网络，促进多部门协作
   - 时间大幅缩短：
     - 冬储分工准备时长：15小时缩短至0.5小时
     - 冬储策划编制：300小时缩短至137小时
     - 任务跟踪与周报撰写：139小时减少至35小时
   - 累计提效：174%

4. **作业安全分析效率提升**
   - 线上化：全面线上化作业安全分析
   - 信息沉淀：自动沉淀和整合信息，形成产线知识
   - 操作便利：语音转文字录入，预制信息选择
   - 自动提醒：自动化流程每日自动提醒
   - 时间节省：单次作业安全分析从20分钟以上缩短至5分钟以内
   - 提效幅度：75%

### 具体数据支撑

- **成本节约**：30万元（生产计划编排年价值）
- **效率提升**：174%（冬储过程累计提效）
