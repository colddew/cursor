# 敷尔佳飞书客户案例深度分析报告

## 1. 客户概况

### 公司基本信息
- **公司名称**: 敷尔佳
- **成立时间**: 2017年
- **上市情况**: 2023年8月1日在深交所创业板上市
- **行业**: 美妆护肤/医疗器械
- **主营业务**: 专业皮肤护理产品的研发、生产和销售
- **产品类别**:
  - 医疗器械类敷料
  - 功能性护肤品
- **企业规模**: 未明确提及，但从上市情况和行业地位判断为大型企业
- **行业地位**: 全国专业皮肤护理产品细分领域领先品牌，被誉为"国品之光"，黑龙江省生物经济领域第一家"独角兽"企业

### 企业文化与愿景
- **使命**: "让世界见证中国之美"
- **愿景**: 成为"世界一流的功效护肤领导者"
- **产品理念**: "科技赋能，品质第一，功效先行"
- **企业文化**: "坦诚清晰 阳光透明"

## 2. 核心挑战分析

### 挑战一：多渠道、多终端、多产品线协同难题
- **具体表现**:
  - 线上渠道涵盖天猫、抖音、京东、快手、拼多多、唯品会、视频号等主要电商平台
  - 线下渠道覆盖医疗机构、美容机构、连锁零售药店、化妆品专营店及大型商超
  - 各渠道产品不同、客户合作模式不同、产品政策不同
  - 销售数据实时汇总困难，缺乏全景概览

### 挑战二：产品全链路协同复杂
- **具体表现**:
  - 产品全链路长，涉及研发、生产、销售等多个环节
  - 协同方多，包括产品、设计、研发、技术、采购、生产等多个部门
  - 难以合理分工、高效协作，业务节点推进缓慢

### 挑战三：生产管理数字化程度低
- **具体表现**:
  - 生产排产信息同步困难
  - 生产设备管理缺乏系统化支持
  - 设备稼动率统计依靠手写记录
  - 设备报修流程依赖口述、微信等传统方式

## 3. 飞书解决方案应用场景

### 3.1 全渠道销售管理

#### 直播带货管理
- **应用产品**: 飞书多维表格
- **解决方案**:
  - 将所有直播间数据汇总在一张表格中
  - 区分各统计维度，通过查找引用进行数据汇总
  - 利用仪表盘展示销售情况
  - 自动化实现自动增加记录及自动通知

#### 达播合作管理
- **应用产品**: 飞书在线文档、云盘、多维表格
- **解决方案**:
  - **合作前**: 通过在线文档审批达人，综合评估知名度、粉丝群体、直播风格
  - **合作中**: 云盘存储录屏、产品资料、样品寄送等数据
  - **合作后**: 多维表格归拢达人信息、合作佣金、GMV、对接人等信息
  - 通过仪表盘识别优质合作达人，为二次合作提供参考

#### 多渠道销售数据整合
- **应用产品**: 飞书多维表格
- **解决方案**:
  - 多人协作功能实现信息上传、完善、维护和更新的全流程覆盖
  - 高级权限设置确保不同部门互不干扰
  - 实现数据的准确性和实时性

### 3.2 AI赋能客户服务

#### 客服知识管理
- **应用产品**: 飞书知识库
- **解决方案**:
  - 内嵌产品词典、业务流程图、处理思路等超链接
  - 定期维护信息资料，提升客服团队工作效率
  - 解决客服人员流动大、培训资料杂乱的问题

#### 消费者评价分析
- **应用产品**: 飞书AI、多维表格
- **解决方案**:
  - 将各平台评价数据汇总于电子表格
  - 使用AI助力评价内容分析，1分钟生成固定公式
  - 2分钟处理上万条评价，提取产品特性、关键词
  - 定期出具产品分析报告，为决策提供参考

### 3.3 线下巡店与溯源管理

#### 数字化巡店
- **应用产品**: 飞书表单、多维表格
- **解决方案**:
  - 移动端随时上传巡店结果及照片
  - 表格仪表盘进行环比、同比分析
  - 基于数据分析下达任务，实现从发现问题到整改的全流程管理

#### 查货溯源管理
- **应用产品**: 飞书多维表格
- **解决方案**:
  - 协同操作，多维度查看所需数据
  - 所有查货情况被沉淀和实时查看
  - 解决多个渠道数据无法同步的问题

### 3.4 产品研发协同

#### 产品设计协同
- **应用产品**: 飞阅会、云文档
- **解决方案**:
  - 使用飞阅会和云文档评论功能
  - 解决文字多、重点内容不突出的问题
  - 实现实时跟进汇报进度

#### 产品研发项目管理
- **应用产品**: 飞书项目
- **解决方案**:
  - 新品研发项目在飞书项目中录入及流转
  - 项目流程清晰透明
  - 延期风险自动触发提醒
  - 实现项目标准化，风险可控化

#### 产品知识培训
- **应用产品**: 飞书知识库、词典
- **解决方案**:
  - 建立产品介绍信息定期维护更新机制
  - 实时更新产品介绍内容并通知全员
  - 建立词典整理细致化问题
  - 将部门业务板块编辑成知识库

### 3.5 生产管理数字化

#### 生产排产计划共享
- **应用产品**: 飞书多维表格、公共日历
- **解决方案**:
  - 清晰了解各车间产品排产信息
  - 简化原表格转换及多表格录入的繁琐流程
  - 运用高级设置权限、自动化功能实现"1分钟"员工确认信息

#### 生产设备管理系统
- **应用产品**: 飞书多维表格、表单、自动化、机器人
- **解决方案**:
  - 搭建设备管理系统（设备管理模块+备件管理模块）
  - 设置专属二维码、个性化定制表单
  - 实现设备日常管理、线上统计开机稼动率
  - 在线培训、实时查看操作文件及标准操作视频
  - 一键式设备报修管理、设备云文档台账
  - 备品备件二级库管理

## 7. 战略洞察与价值分析

### 7.1 数字化转型成功要素
1. **全链路思维**: 从研发到销售的全链路数字化，而非单一环节优化
2. **轻量化实施**: 通过飞书平台快速搭建轻量业务系统，避免重型系统开发
3. **AI赋能**: 充分利用AI能力提升数据分析效率
4. **移动优先**: 重视移动端体验，支持一线业务需求
5. **知识沉淀**: 建立系统化知识管理，避免知识流失

### 7.2 美妆行业数字化启示
1. **渠道管理复杂性**: 多渠道销售是美妆行业的特点，需要统一的数据管理平台
2. **快速响应市场**: 消费者反馈的快速分析对产品迭代至关重要
3. **合规性要求**: 医疗器械类产品有特殊合规要求，数字化流程需要考虑合规性
4. **达人经济管理**: 直播带货和达人合作需要系统化管理

### 7.3 对同类企业的借鉴意义

#### 适合采用类似模式的企业特征
- **多渠道销售企业**: 线上线下多渠道销售的企业
- **产品多样化企业**: 产品线复杂，需要协调管理的企业
- **快速迭代行业**: 需要快速响应市场变化的行业
- **协同复杂企业**: 跨部门协作复杂的企业
