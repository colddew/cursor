# 医家通飞书客户案例分析报告

## 1. 公司基本信息

**行业领域**：医疗健康/智慧养老
**企业性质**：初创企业
**主营业务**：整合社区养老、居家养老、机构养老、医养结合等多种养老场景，提供全场景养老管理体系与信息管理系统
**服务规模**：为全国1500+养老机构提供智慧养老方案
**业务特点**：为不同养老单位提供丰富的、多维度的智能养老解决方案

## 2. 飞书应用场景

### 2.1 目标管理与组织协同
- **飞书OKR**：建立公开透明的组织文化，让组织目标清晰一致
- 部门间目标联动，解决各部门"闭门造车"问题
- 每周开展OKR碰头会，确保组织聚焦同一目标

### 2.2 客户信息管理
- **飞书多维表格**：整合1500+客户资料，高效归整客户信息
- 建立以飞书文档为形式的知识库和部门手册
- 实现对客户核心信息快速索引，多维度、多界面、多形式呈现客户案例关键信息

### 2.3 会议协同
- **飞书会议**：替代多个会议工具，解决日程同步繁琐、切换不流畅问题
- **飞阅会模式**：采用"先阅读后讨论"模式，以飞书文档作为会议信息传递载体
- **妙记功能**：结合飞书文档形式进行会议纪要

### 2.4 一体化办公
- **All in One解决方案**：替代数十种办公工具
- 集成飞书日历、飞书会议室等功能，提供统一办公体验
