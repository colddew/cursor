# 新松机器人飞书客户案例分析报告

## 2. 飞书给客户带来的新能力和价值

### 2.1 具体使用的飞书功能模块

1. **飞书多维表格**
   - 用于物料盘点管理（每个BG一张多维表格）
   - 用于硬件交付项目全生命周期管理（3张多维表格）
   - 自动生成仪表盘用于数据分析

2. **飞书知识库**
   - 企业知识沉淀和管理
   - 信息体系化存储

3. **飞书订阅号**
   - 企业信息发布
   - 安全宣传（新松安全广播号）

4. **飞书文档**
   - 协作编辑和文档管理

5. **飞书即时通讯(IM)**
   - 自动通知功能
   - 团队协作沟通

### 2.2 解决的业务痛点

1. **物料盘点管理问题**
   - **之前**：纸质表管理，各BG信息收集归总效率低，数据统一维护难，缺少数据分析
   - **解决方案**：每个BG一张多维表格加一张仪表盘，盘点过程高效、安全、有保障
   - **成果**：1个仪表盘掌握物料盘点全局情况

2. **多项目进度管理问题**
   - **之前**：硬件交付项目多，环节多，缺乏趁手的管理工具；线下表格管理，接力编辑效率低，数据安全难保障，经验难沉淀
   - **解决方案**：用飞书多维表格管理全部项目进展，灵活搭建，轻量运营
   - **成果**：3张表覆盖硬件交付生命周期管理

3. **知识管理问题**
   - **之前**：文件储存在不同本地硬盘，管理难，权限把控难，无法形成有效知识沉淀
   - **解决方案**：用飞书知识库和订阅号让企业信息体系化，知识更好地沉淀
   - **成果**：8个集团知识库，有效知识沉淀

### 2.3 带来的业务价值和效率提升

1. **物料盘点效率提升**
   - 从纸质表格转为线上无纸化盘点
   - 实时数据更新和自动化推送通知
   - 自动生成仪表盘，管理者可掌握全局情况
   - 权限控制保障信息安全

2. **项目管理轻量化**
   - 避免使用沉重的系统，飞书补齐"数字化最后一公里"
   - 多人协同录入信息，操作方便
   - 权限隔离方便管控，数据更安全
   - 阶段变化自动通知相关人员

3. **知识资产化管理**
   - 企业信息体系化，信息上云
   - 提升信息流转时效性、敏捷性
   - 提升无形资产管理效率
   - 打造知识共创共享的企业文化
   - 促进员工自驱成长

4. **安全管理创新**
   - 通过视频形式进行安全宣导
   - 内容为主导进行安全普及渗透
   - 安全科普内容有效沉淀
   - 安全理念深入人心

## 3. 飞书未能满足的客户需求

基于现有案例内容，分析得出以下可能未满足的需求：

### 3.1 深度业务系统集成需求
- 案例中提到新松选择了"不是沉重的、复杂的系统"，暗示飞书在深度业务系统集成方面可能有所限制
- 作为高端装备制造企业，可能需要与生产执行系统(MES)、企业资源规划(ERP)等深度业务系统进行更紧密的集成

### 3.2 专业研发工具支持
- 作为拥有4000余人研发团队的高科技企业，可能需要更专业的研发项目管理工具
- 对于复杂的机器人研发项目，可能需要更专业的版本控制、代码管理、测试管理等工具集成

### 3.3 全球化协作支持
- 产品出口40多个国家和地区，为4000余家国际企业提供服务
- 可能需要更强的多语言支持、跨时区协作、跨国数据合规等功能

### 3.4 行业特定功能
- 作为机器人制造商，可能需要特定的工业互联网、物联网设备管理、远程监控等功能
- 制造业特有的设备维护、质量管理、供应链协同等专业化需求

### 3.5 高级数据分析能力
- 虽然提到了物料盘点数据分析，但作为高科技制造企业，可能需要更高级的生产数据分析、预测性维护、质量控制分析等AI驱动的分析功能

### 3.6 合规与安全需求
- 作为上市公司和央国企，可能需要更严格的数据安全、访问控制、审计追踪等功能
- 制造业特有的知识产权保护、商业秘密管理等高级安全需求

## 4. 战略洞察与建议

### 4.1 飞书在制造业的价值定位
飞书在新松机器人的成功应用表明，其在制造业的价值主要体现在：
1. 轻量化的业务流程管理，填补"数字化最后一公里"
2. 知识管理和协作效率提升
3. 灵活的项目管理能力，适应复杂多变的项目环境

### 4.2 制造业数字化转型关键点
1. **渐进式数字化**：从轻量级工具入手，避免一开始就投入复杂系统
2. **知识资产化管理**：重视无形资产的沉淀和管理
3. **流程标准化与灵活性平衡**：既要有标准化流程，又要保持灵活性

### 4.3 对飞书的战略建议
1. **加强行业深耕**：针对制造业特定场景开发更多专业化功能
2. **深化系统集成能力**：提供与更多制造业核心系统的深度集成方案
3. **提升全球化支持**：加强多语言、跨时区、跨国合规等功能
4. **AI赋能**：将更多AI能力应用于制造业场景，如预测性维护、质量检测等
