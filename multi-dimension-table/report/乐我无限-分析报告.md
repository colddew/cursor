# 乐我无限飞书客户案例分析报告

## 1. 客户基本信息

### 公司名称
乐我无限

### 所属行业
互联网/泛娱乐行业，专注于出海业务

### 公司规模
- 成立于2016年，至今运营超过5年
- 2019年9月从猎豹移动拆分，完全独立运营
- 被描述为"小而美"的公司，专注于人才密度而非规模扩张

### 主营业务
- 主要产品：LiveMe（自2016年4月登陆海外应用市场）
- 业务领域：社交、直播、游戏等泛娱乐领域
- 服务范围：用户遍布220多个国家和地区
- 公司愿景："连接世界，传递快乐"
- 定位：海外泛娱乐产品矩阵的出海服务商

## 2. 飞书给客户带来的新能力和价值

### 具体使用的飞书功能模块

1. **飞书招聘核心功能**
   - 三合一工作台（简历、面试、面评在同一页面）
   - 视频会议集成
   - 移动端应用
   - 结构化面试工具（"面试助手"）
   - 自动化Bot提醒系统

2. **配套使用模块**
   - 飞书视频会议
   - 飞书文档
   - 飞书日历
   - 飞书会议室

### 解决的业务痛点

1. **招聘效率问题**
   - 解决了特定岗位高端人才稀缺带来的招聘竞争激烈问题
   - 提高了出海企业对语言能力要求高的招聘效率
   - 解决了业务部门要人积极但执行消极的问题

2. **多地区协作挑战**
   - 解决了跨时区、跨地域的招聘协作问题
   - 实现了随时随地处理招聘关键工作的能力
   - 减少了因时差造成的流程卡点

3. **面试质量标准化**
   - 解决了业务方面试官众多但无法严格做到结构化面试的问题
   - 统一了面试标准和流程
   - 提高了面试评估的公正性和完整性

### 带来的业务价值和效率提升

1. **招聘流程效率提升**
   - 面试官从工作任务中无缝切换到招聘系统
   - 飞书Bot自动发送面试日程邀请、面试提醒、面评填写提醒
   - 最大程度节省沟通成本，规范化招聘流程
   - 面试官可以当场提交面评，更快将候选人推到下一环节

2. **移动办公能力**
   - 面试官可在碎片化时间处理招聘相关工作
   - 如在回家路上开启面试、审批Offer
   - 移动端与PC端数据同步，一键触达任务页面

3. **结构化面试质量提升**
   - 面试助手推送参考面试问题，提示岗位考核点
   - 面试官可围绕每道题目记录候选人回答并给出评估结果
   - 基于字节跳动数以百万计的面试积累，提供专业的面试指导

4. **招聘体验优化**
   - 打造愉悦的招聘体验，减少流程效率黑洞
   - "把琐碎交给工具，把思考留给人"的设计理念
   - 一体化设计让面试环节变得非常顺畅
