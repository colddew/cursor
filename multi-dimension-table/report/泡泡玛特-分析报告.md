# 泡泡玛特飞书客户案例分析报告

## 1. 公司基本信息

### 行业定位
- **行业**：潮流文化娱乐行业/潮玩行业
- **成立时间**：2010年（14年历史）
- **品牌理念**："创造潮流，传递美好"
- **核心业务**：IP孵化与运营、商品开发与零售、主题乐园与体验、娱乐四大领域

### 业务规模
- **全球门店**：450+家线下门店
- **机器人商店**：2300+台
- **市场覆盖**：全球80+个国家和地区
- **粉丝规模**：3000万+全球粉丝
- **知名IP**：MOLLY、SKULLPANDA、DIMOO、THE MONSTERS、HIRONO等
- **财务表现**：
  - 头部IP SKULLPANDA、MOLLY收入均突破10亿人民币
  - 抽盒机2023年收入超7亿人民币
  - 抖音平台2023年收入超2.8亿人民币，同比增长超400%

## 2. 飞书应用场景

### 2.1 IP孵化与运营管理
- **产品开发进度管理**：使用飞书多维表格管理IP孵化运营全流程，确保全链路所有角色清晰了解产品开发进展
- **信息安全保护**：利用飞书云文档保密能力，对IP孵化及运营全流程信息进行权限管控与下载记录留痕
- **跨语言协作**：与海外艺术家沟通时，使用飞书实时翻译功能，各自说母语即可完成无障碍沟通
- **知识管理**：通过飞书知识库沉淀产品信息，形成产品说明小册子，赋能一线门店/客服团队，包括IP名称录音库

### 2.2 门店零售管理
- **门店巡检**：每日对全国300+家门店进行20+个检查项的详细巡检，覆盖卫生管理、服务水平、货品管理、陈列管理等
- **巡检流程线上化**：通过飞书定制专属巡检小程序，移动端灵活操作
- **问题整改管理**：支持"随手拍问题"、@相关负责人、创建整改问题、分享到聊天和催促整改等功能
- **协同跟进**：多部门协调推进时，参与者可随时跟进整改进展

### 2.3 抽盒机产品研发
- **产研协同标准化**：使用飞书项目实现复杂产研协同流程的标准化、高效化
- **流程可视化**：每个需求可查看流程进展、责任人等
- **协作高效化**：流程节点灵活设置，@拉人进项目，需求进展自动同步到群
- **管理精细化**：在线管理研发人员排期，清晰掌握几十名研发的项目进度和时间安排

### 2.4 抖音直播运营
- **排班管理**：使用飞书多维表格管理20+名场控、30+名主播的直播排期
- **日程自动提醒**：直播开始前1天，系统自动为主播创建飞书日程并发送消息提醒
- **一表管理档期**：所有直播计划沉淀在一张表上，支持不同视角创建日历、看板等视图
- **效果复盘**：在线提交交易额、场观人数、转化率等关键信息，快速对直播效果做复盘

### 2.5 全球化协作
- **跨语言协作**：支持不同语言员工无障碍沟通
- **跨时区协作**：解决不同时区员工的协作问题
- **跨文化协作**：促进不同文化背景员工的高效协作

### 2.6 乐园设计施工管理
- **设计审核管理**：通过飞书审批结合云文档能力，管理大量设计文件的审核和变更流程
- **外包协作**：与大量外包商、设计院对接，高频审核文件
- **流程管理**：管理整个项目流程从发起、审批到归档的完整项目周期
