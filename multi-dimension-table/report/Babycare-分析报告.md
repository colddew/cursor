# Babycare飞书客户案例分析报告

## 1. 公司基本信息

**公司名称**：Babycare

**行业领域**：母婴行业

**公司规模**：国际化企业，在30多个国家开展业务

**服务用户规模**：超过5000万用户

**主营业务**：一站式、全品类的母婴产品解决方案

**公司使命**：以"秉父母之心做产品"为使命，不断发现并改变行业中存在的不合理，为爱重新设计，定义Z世代人群新的孕婴童生活方式

**公司特点**：设计师创立的母婴品牌，覆盖线上电商平台、线下门店以及社交电商等多渠道销售模式

## 2. 飞书应用场景

Babycare将飞书作为数字化底座，构建了高效的移动工作台，主要应用于以下四个核心业务场景：

### 2.1 全渠道客服质检管理
- **应用产品**：飞书多维表格、飞书文档
- **应用范围**：覆盖线上电商平台、线下门店和社交电商等多渠道
- **管理规模**：十几个人完成几百个客服的质检管理

### 2.2 异常件处理流程管理
- **应用产品**：飞书多维表格、飞书招聘、飞书绩效
- **业务范围**：退款维权、物流异常等特殊事件处理
- **协同部门**：退款维权部、客服、仓库等部门

### 2.3 合同全生命周期管理
- **应用产品**：飞书合同、飞书会议室
- **应用范围**：法务、财务等部门协同工作
- **管理内容**：合同风险检索、台账管理、到期提醒、业财数据一体化

### 2.4 社群运营数据管理
- **应用产品**：飞书多维表格、即时信息、飞书服务台
- **应用范围**：私域运营部门
- **管理内容**：用户增长、互动、转化等数据统计与分析

## 3. 新获得的能力

### 3.1 数字化协同能力
- **移动工作台构建**：以飞书为底座构建了移动工作台，实现高效协同
- **流程在线化**：实现业务流程在线管理和跟踪
- **数据互通**：打破数据孤岛，实现跨部门数据实时共享
- **跨组织互动**：支持多部门、多角色的高效协作

### 3.2 客服管理数字化能力
- **质检效率提升100%**：客服质检效率提升100%，从原来需要2小时以上缩短至不到1小时
- **自动化质检流程**：实现质检流程自动化和数据集中管理，减少手动记录和数据汇总时间
- **实时反馈机制**：质检结果和改进建议得以实时传达，显著提升客服质量和客户满意度
- **数据驱动决策**：集中管理的客服数据提供全面分析，帮助管理层识别培训需求并开展针对性培训

### 3.3 异常处理能力
- **处理周期缩短**：异常件处理周期从3天缩短至1天，效率提升66.7%
- **信息快速传递**：确保信息迅速传递，相关人员可实时沟通
- **实时监控能力**：管理层可以通过仪表盘实时关注资损追款和退款维权的整体情况
- **科学决策支持**：通过图表展示关键指标，支持做出科学决策和资源调配

### 3.4 合同管理能力
- **全流程管控体系**：从0到1搭建起合同全生命周期管控体系
- **风险智能识别**：自动提示风险，实时展示工商信息、交易方签约历史信息等内容
- **数据资产化管理**：合同台账系统自动生成，按需自定义产品类型、店铺类型等信息
- **业财数据一体化**：关联财务预算与付款流程，打破数据孤岛，实现业财数据实时准确传输
- **智能提醒机制**：合同履约数据清晰，系统自动提醒相关人员，确保及时跟进，避免遗漏

### 3.5 社群运营能力
- **跨平台数据集中管理**：实现私域运营数据跨平台管理
- **数据自动化统计**：系统自动汇总用户增长、互动、转化等数据，提高统计效率和准确性
- **实时监控反馈**：确保对社群动态的快速响应
- **可视化分析**：数据可视化功能提供直观的数据展示，助力做出更高效和精准的运营决策

## 4. 未满足的需求

### 4.1 系统集成局限性
- **数据触达率低**：在使用飞书前，业务全流程通过在线客服系统完成，需要登陆平台获取数据，数据触达率低
- **提醒机制不足**：数据变动提醒不能直接触达员工，流转效率低，重要事项难以及时跟进

### 4.2 人工操作依赖
- **人工催促跟进**：异常件处理需与仓库协同，客服需人工催促物流单号
- **手动检索耗时**：法务需手动检索风险及合作历史，耗时且风险信息不实时
- **人工提醒易遗漏**：合同到期需人工提醒，易遗漏

### 4.3 数据管理挑战
- **数据分散**：私域运营数据分散，无统一查看渠道
- **缺乏可视化工具**：数据可视化工具缺失，增加数据汇总和分析的工作量
- **台账管理繁琐**：合同台账管理繁琐，易出错

### 4.4 流程标准化需求
- **多渠道标准不统一**：客服质检多渠道标准不统一，影响客服质量一致性
- **审批流程缺乏参考**：审批流程缺乏高效参考，影响合同管理效率
