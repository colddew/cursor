## 2. 飞书给客户带来的新能力和价值

### 2.1 具体使用的飞书功能模块

1. **飞书知识库**: 打造企业级知识库，用于存储标准化物料和学习资料
2. **飞书多维表格**: 用于数据管理和协作
3. **飞书文档**: 用于文档协作和信息共享
4. **即时消息**: 用于日常沟通和信息传递
5. **飞书日历**: 用于日程安排和会议管理
6. **音视频会议**: 支持远程协作和"云火锅"等特殊场景
7. **飞书绩效**: 用于绩效管理
8. **会议功能**: 特别是"飞阅会"模式，提高会议效率

### 2.2 解决的业务痛点

1. **知识管理痛点**: 
   - 之前没有统一的、成体系的知识库
   - 业务、产品、战略等核心知识依赖人工和传统线下会议传递，效率低下

2. **信息流动痛点**:
   - 基层员工的创新灵感难以自下而上传递
   - 跨部门沟通不畅
   - 宣传材料效果难以追踪

3. **会议效率痛点**:
   - 传统"异步协作"会议模式效率不高
   - 一个人讲，其他人等着，无法同时沟通

### 2.3 带来的业务价值和效率提升

1. **知识管理提升**:
   - 建立了企业级知识库，为需求较多的销售等部门提供便利
   - 打造了"学习中心"，沉淀新员工培训和主题培训内容
   - 提升了知识管理水平和组织内部协同效率

2. **信息流动优化**:
   - 创建了信息通道，让基层员工灵感可以自下而上快速传递
   - 云文档可以轻松分享并授权协同编辑，促进跨部门沟通
   - 宣传材料搬到飞书云文档上，能够追踪反馈和效果

3. **会议效率提升**:
   - "飞阅会"模式变"异步协作"为"同步协作"
   - 会前阅读文档并添加评论，会上只聚焦未解决的评论
   - 快速得出结论，大大提升会议效率

4. **协作体验改善**:
   - 沟通、日历、文档、音视频等功能整合到一个平台
   - 不再需要在各个模块间切换工具，一站式体验节省时间
   - 快捷表情回复不会打扰他人，表态更积极
