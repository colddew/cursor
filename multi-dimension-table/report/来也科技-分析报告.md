# 来也科技飞书客户案例深度分析报告

## 1. 客户基本信息

### 公司名称
来也科技

### 所属行业
智能自动化科技行业

### 公司规模
- 员工数量：超过700人
- 团队分布：在中国大陆和全球多地设有办公室
- 海外团队：超过100人的海外团队

### 主营业务
来也科技是全球领先的智能自动化领军品牌，为客户提供变革性的智能自动化解决方案，主要包括：
- 机器人流程自动化（RPA）
- 智能文档处理（IDP）
- 对话式AI（Conversational AI）
- 流程挖掘（Process Mining）

公司服务客户涵盖保险、通信、电力、金融、零售、医疗等众多行业。2021年《财富》世界500强榜单前十名企业中，7家在使用来也科技的智能自动化产品。2022年4月，公司完成了7000万美元C++轮融资。

## 2. 飞书给客户带来的新能力和价值

### 具体使用的飞书功能模块

1. **飞书OKR目标管理**
   - 构建目标网络，实现制定、跟进、复盘的闭环管理
   - 提供OKR填写助手，提示"低质量"风险
   - 支持在飞书文档中插入OKR
   - 一键翻译功能，支持多语言团队协作

2. **飞书文档**
   - 作为OKR制定和复盘的工作平台
   - 周会会议文本载体
   - 支持内容一键回传到OKR系统

3. **飞书绩效管理系统**
   - 搭建符合公司需求的绩效流程
   - 实现360环评和上级评估
   - 与飞书OKR系统打通
   - 内置功能报表快速获取关键信息
   - "手松手紧"体系，确保评估客观性

### 解决的业务痛点

1. **目标管理挑战**
   - 业务迅速发展、团队不断壮大
   - 如何对齐全员目标、激发员工突破和创新
   - 多元团队（来自不同国家）如何高效完成OKR的制定、对齐、跟进

2. **绩效管理痛点**
   - 使用持续的及时绩效反馈需求
   - 高频的评估节奏（季度评估）
   - 相对复杂的评估环节（OKR参考、360评估等）
   - 传统绩效评估工具效率低下，需要"发表格、填表格、收表格、汇总表格"

3. **组织透明度问题**
   - 提升组织战略透明度
   - 公平地激励优秀人才
   - 激发团队创新

### 带来的业务价值和效率提升

1. **OKR管理效率提升**
   - 全员普及OKR管理，新员工第一天即接受OKR培训
   - 周期性检查业务部门OKR制定和复盘质量
   - 通过飞书订阅号广而告之高质量OKR案例

2. **绩效评估流程优化**
   - 从半年一次恢复为季度评定机制
   - 整个绩效评定时间控制在一周内完成
   - 线上流转，省去HR和部门管理者"收发"动作
   - 信息透明，参与评估者无需切换系统即可了解被评估人的目标和产出

3. **多语言团队协作**
   - 通过一键翻译功能，不同文化背景员工高效了解彼此
   - 外籍员工更好地建立企业归属感
   - 员工可以用母语表达想法

### 具体数据支撑

1. **绩效评估时间优化**
   - 曾经因时间成本考虑将季度评估改为半年评估
   - 现在不仅恢复季度评估，还能在不影响质量前提下将整个流程控制在1周内

2. **团队规模管理**
   - 成功管理超过700人的员工团队
   - 支持多国家、多地区的团队协作
   - 超过100人的海外团队高效协作

3. **培训体系效果**
   - 新员工入职第一天接受OKR介绍
   - 入职一周接受系统OKR培训
   - 新员工培训中OKR制定和复盘成为专项课程

### 潜在的功能需求

1. **更深度的行业定制化**
   - 作为智能自动化行业的领军企业，可能有更多行业特定的管理需求
   - 可能需要更专业的项目管理模板，适配软件开发和自动化解决方案交付

2. **全球合规性支持**
   - 作为跨国公司，可能需要更多关于不同国家数据保护法规的合规性支持
   - 可能需要更完善的跨国数据治理和权限管理功能

3. **AI能力集成**

   - 可能希望将自家的AI产品更深度地集成到飞书平台中

### 仍需其他解决方案的领域

1. **专业业务系统**

   - 飞书主要解决的是办公协作和人力资源管理问题

2. **开发工具链**
   - 作为科技公司，仍需要专业的开发工具、版本控制系统、CI/CD流水线等
   - 这些不是飞书的核心功能

3. **客户服务平台**
   - 需要专业的客户服务和支持系统来管理客户关系
   - 飞书可能无法完全替代专业的CRM系统

## 4. 战略洞察与建议

### 飞书的核心价值定位
从案例中可以看出，飞书在来也科技的成功应用主要体现在：
1. 提供了完整的组织管理和人才发展解决方案
2. 通过OKR和绩效管理实现了组织目标的高效对齐
3. 支持了多语言、多地域团队的协作需求
4. 将传统人力资源管理流程数字化、效率化

### 飞书的机会空间
1. **行业深度整合**：可以与更多专业软件公司合作，将通用办公平台与专业业务系统深度整合
2. **全球化支持**：继续加强多语言、多时区、多法规环境下的企业协作支持
3. **AI能力增强**：作为智能自动化公司的合作伙伴，可以探索更深度的AI能力集成

### 对其他企业的启示
1. OKR管理需要与企业文化深度融合，需要从新员工入职开始就培养相关意识
2. 绩效管理的数字化转型可以显著提升效率，但需要设计符合企业特点的流程
3. 多元化团队的管理需要考虑语言、文化等因素，合适的工具可以起到关键作用
