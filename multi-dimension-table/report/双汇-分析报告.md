# 双汇飞书客户案例分析报告

## 1. 公司基本信息

**行业：** 食品制造业（全产业链食品企业）

**规模：** 数万名员工的大型企业

**主营业务：** 肉类食品生产与销售，涵盖从原料采买、工厂生产到全国数十万销售网点的全产业链

**核心业务特点：**
- "从农场到舌尖"的完整食品产业链
- 生鲜品和肉制品等多产品线运营
- 全国数十万销售网点的分销网络

## 2. 飞书应用场景

### 2.1 组织协同平台构建
- **"汇工作"统一平台**：集成上下游产业链数十个系统入口，实现统一登录体验
- **内部社区"汇友圈"**：员工交流平台，处理就餐、停车、差旅等日常问题，改善员工体验
- **跨部门协作**：覆盖采购、财务、研发以及各事业部的数字化应用

### 2.2 业务流程数字化
- **无纸化改造**：将50+纸质报表转化为飞书多维表格
- **工厂日报系统**：生产数据实时填报与汇总
- **安全管理升级**：安全检查流程数字化，1000+检查项上线

### 2.3 数据驱动决策
- **仪表盘体系**：实时监控养殖进度、生产数据、销售情况
- **多环节数据分析**：采购高点低点分析、产量预测、销售业绩追踪
- **自动化报表**：定时推送、定时回收数据，当日形成经营预测
