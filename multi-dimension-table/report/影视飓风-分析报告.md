# 影视飓风飞书客户案例分析报告

## 1. 客户基本信息

### 公司名称
影视飓风 MediaStorm

### 所属行业
文娱传媒、视频制作行业

### 公司规模
- 成立于2015年
- 2019年团队成长到20多人的规模
- 目前B站平台粉丝数超440万
- 年产近100期原创视频（不含商业广告摄制等业务）

### 主营业务
- 视频教学
- 器材评测
- 优质原创短片制作
- 商业广告摄制
- 影视资源分享

### 公司荣誉
连续三年被评为"bilibili百大UP主"

## 2. 飞书给客户带来的新能力和价值

### 2.1 具体使用的飞书功能模块

1. **飞书多维表格**
   - 搭建创意选题库
   - 项目进度管理
   - 任务分配与追踪
   - 员工表现量化

2. **飞书云文档**
   - 标准化工作流程文档
   - 项目专属文档创建
   - 文档间嵌套与关联
   - 丰富元素插入（流程图、思维导图等）

3. **飞书知识库**
   - 失败经验整理沉淀
   - 《那些年影视飓风翻过的车》知识文档
   - 新人入职指南

4. **飞书OKR**
   - 目标管理与追踪
   - 团队目标对齐

5. **飞书招聘**
   - 人才招聘管理

6. **飞书绩效**
   - 员工绩效管理

7. **飞书会议室**
   - 会议管理

### 2.2 解决的业务痛点

1. **爆款内容创作效率低下**
   - 原问题：平均需要6-7个月才会出现一期播放量达百万的视频
   - 解决方案：用飞书多维表格集成创意点子，摸清优质内容输出路径

2. **项目管理混乱**
   - 原问题：缺乏关联性任务协同推进，管理者难以统揽全局，项目流转周期超过35-40天
   - 解决方案：借助任务面板、进展图等功能统揽全局，量化员工表现

3. **信息沉淀分散**
   - 原问题：员工在不同办公工具间切换，创意脑暴无法专注，讨论结果分散在各自文档
   - 解决方案：用飞书云文档标准化工作流程，构建组织内部庞大信息网络

4. **经验传承困难**
   - 原问题：项目执行过程中出现的问题没有很好地归整沉淀，新人入职后依旧"踩坑"
   - 解决方案：将知识库与多维表格打通，整理失败经验，明确责任人

### 2.3 带来的业务价值和效率提升

1. **内容产出效率大幅提升**
   - 从原来6-7个月才有百万播放量视频，提升到每隔15天就有一期超100万播放量视频
   - 效率提升**10倍**以上

2. **项目流转周期大幅缩短**
   - 项目平均流转周期从原有的35-40天降低到**5天**
   - 项目平均流转周期降低**85%**

3. **组织管理标准化**
   - 所有小磁铁吸到一起变成规则形状
   - 让组织更有方向感、工作更加标准化

4. **知识管理能力增强**
   - 形成庞大的云端文档网络
   - 构建信息枢纽中心
   - 避免重复犯错，降低试错成本

### 2.4 具体数据支撑

1. **10倍** - 视频产出效率提升，从6-7个月到15天
2. **85%** - 项目平均流转周期降低，从35-40天到5天
