# 联影医疗飞书客户案例分析报告

## 执行摘要

本报告深入分析了联影医疗作为飞书客户的数字化转型案例。联影医疗作为中国高端医学影像装备领域的龙头企业，通过飞书平台构建了全面的数字化底座，成功解决了跨学科协作、专业知识管理和复杂研发流程等核心挑战。该案例对医疗器械行业乃至其他高科技制造业的数字化转型具有重要的借鉴意义。

## 1. 公司基本信息

### 1.1 公司概况
- **公司名称**: 上海联影医疗科技股份有限公司
- **行业定位**: 医疗健康/医疗器械/高端医学影像装备
- **业务领域**: 科技、先进制造、医疗三大战略交叉领域
- **成立时间**: 2011年
- **发展历程**: 12年间推出多款「世界首创、行业首创」产品

### 1.2 主营业务
- 全线自主研发的高性能医学影像诊断与治疗设备
- 生命科学仪器
- 覆盖「基础研究-临床科研-医学转化」全链条的创新解决方案

### 1.3 企业愿景与使命
- **愿景**: 成为世界级医疗创新引领者
- **核心驱动**: 创新是业务发展的首要驱动因子
- **合作模式**: 与全球高校、医院、研究机构及产业合作伙伴深度协同

### 1.4 员工构成特点
- **学科背景**: 来自40多个学科专业背景的专业人才
- **专业性**: 人员专业度高，学科知识十分繁复
- **国际化**: 全球化业务不断开展

## 2. 飞书应用场景

### 2.1 数字化转型定位
飞书作为联影医疗的**数字化底座**和**员工一站式信息门户**，全面赋能企业运营。

### 2.2 核心应用功能

#### 2.2.1 跨学科协作与术语管理
- **飞书词典**: 整合所有专业术语，与业务场景紧密结合
- **跨语言翻译**: 结合AI算法能力，提升医疗术语翻译准确性
- **实时释义**: 聊天、看文档、开会时专业术语一键显示释义与相关链接

#### 2.2.2 专业知识管理
- **飞书知识库**: 为研发团队提供项目技术、法规、文档撰写等知识
- **飞书多维表格**: 为设计团队打造"创新平台"管理创意灵感
- **飞书服务台**: 为售后团队提供机器问题解决方案库

#### 2.2.3 研发项目管理
- **飞书项目**: 可视化流程图，将医疗器械研发复杂流程有序拆解
- **飞书会议室**: 支持研发站会项目汇报
- **云文档**: 简化汇报流程，省去额外做PPT时间

### 2.3 部门定制化解决方案

| 部门 | 主要工具 | 应用场景 | 核心价值 |
|------|----------|----------|----------|
| 设计团队 | 多维表格 | 创新平台，管理创意灵感 | 抓住转瞬即逝的灵感，促进跨部门共创 |
| 研发团队 | 知识库 | 项目深度培训，技术文档管理 | 唯一信息获取入口，确保材料及时更新 |
| 售后团队 | 服务台 | 维修问题快速定位与解决 | 93%问题解决率，提升现场维修效率 |

## 3. 新获得的能力

### 3.1 跨学科协作能力
- **术语壁垒打破**: 40多种学科背景人才畅快交流，实现同频共振
- **沟通成本降低**: 专业术语一键释义，极大降低搜索学习成本
- **多语言协作**: 跨语言翻译准确度提升，开会时间降低20%以上

### 3.2 知识管理能力
- **知识资产最大化**: 针对不同部门制定定制化知识管理方案
- **信息触达精准化**: 在合适时间以恰当方式触达正确部门
- **知识融入日常工作**: 知识自然融入员工日常办公细节

### 3.3 研发管理能力
- **流程可视化**: 像给复杂医疗设备研发流程拍CT图，各节点要求一目了然
- **敏捷开发**: 快速迭代，让医疗器械研发这头"大象"也能轻盈起舞
- **汇报效率提升**: 一张图简洁明了，释放研发人员生产力

### 3.4 组织协同能力
- **一站式信息门户**: 员工统一信息获取平台
- **跨部门共创**: 上下游部门就同一张表共创协作
- **创新加速**: 模糊创新思考在产品侧的实际落地和打磨

### 3.5 量化成果指标
- **93%**: 飞书服务台帮助售后团队解决维修问题率
- **20%+**: 多语言团队开会时间降低比例
- **5个月**: 从签约到产生十几份最佳实践的时间
- **1张**: 可视化飞书项目流程图实现复杂流程有序拆解
