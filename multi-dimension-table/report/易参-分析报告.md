# 易参飞书客户案例深度分析报告

## 1. 公司基本信息

### 公司名称
易参（INSSENT）

### 成立时间
2017年

### 行业属性
股权服务行业（法律服务+互联网）

### 公司规模
已服务1500+企业客户

### 投资背景
由保利资本、Carta、腾讯、顺为资本、源码资本等国内外顶级投资机构支持

### 主营业务
一站式股权激励及管理平台，致力于用股权激发人的价值

### 业务特点
- 典型的法律服务与互联网结合的公司
- 业务天然带有专业门槛，市场上现成人才很少
- 基本上都需要自己培养人才
- To B企业服务公司，客户决策周期长

## 2. 飞书应用场景

### 2.1 多维表格管理场景
- **客户数据管理**：整合1500+企业客户的多维度数据
- **运营数据分析**：月会、季度会中的各部门数据整合
- **客户画像可视化**：按行业、轮次、地域等维度划分客户
- **OKR管理**：各部门关键数据清晰呈现，从看板视图到数据报表
- **业务赋能**：帮助业务部门快速获得数据赋能，找到可复用物料

### 2.2 知识库与文档管理场景
- **企业知识库建设**：搭建统一体系化的知识库
- **新人培训**：制作《股权激励基础概念》和《股权激励常见问题手册》
- **部门知识沉淀**：市场、销售、咨询、CSM、产研、人力财务等部门培训资料沉淀
- **个人说明书**：核心团队撰写个人说明书
- **飞书词典应用**：从"人找信息"向"信息找人"转变

### 2.3 会议与沟通场景
- **客户会议**：使用飞书妙记+视频会议记录客户沟通
- **内部分享**：沉淀公司内部分享和直播内容
- **飞阅会实践**：尝试飞书提倡的工作方式
- **协作沟通**：引导客户使用飞书进行协作

## 3. 新获得的能力

### 3.1 数据管理与分析能力
- **数据处理速度提升10倍**：从分散平台到多维表格整合
- **多维度数据分析能力**：可按行业、轮次、地域等维度分析客户
- **实时决策支持**：管理者可基于实时数据做出决策
- **跨部门数据共享**：各部门可快速获得数据赋能
- **优秀经验复制**：找到可复用的物料或优秀的项目服务经验

### 3.2 知识管理与传承能力
- **系统化知识库**：从分散本地文档到统一知识库
- **隐性知识显性化**：将专业顾问脑中的know-how沉淀为文档
- **培训成本降低50%**：通过知识管理替代重复沟通
- **自主学习能力**：员工可随时查看获得支持
- **碎片化学习**：集体培训分散成日常的碎片化学习

### 3.3 会议与协作能力
- **会议内容全记录**：声音、共享画面、转文字全部记录
- **会议效率提升**：员工专注讨论，不用费心记录
- **信息零遗漏**：关键信息不会被遗漏
- **会议成果沉淀**：会议记录可编辑、导出为文档进行二次编辑
- **客户沟通人效提升100%**：高效且准确地完成每个业务环节

### 3.4 组织文化建设能力
- **工作方式升级**：培养飞阅会、写博客、上级给下级写周报等习惯
- **组织同频**：提高协作效率的同时与团队保持同频
- **从"人找人"到"人找信息"**：提高组织效率
- **从"人找信息"到"信息找人"**：持续提高组织效能

## 4. 未满足的需求

### 4.1 行业专业需求
- **专业人才稀缺**：市场上现成人才很少，需要自己培养
- **专业门槛高**：业务天然带有专业门槛，新人培训成本高
- **行业特殊性**：股权服务行业的专业特殊性需要定制化解决方案

### 4.2 客户管理需求
- **客户决策周期长**：To B企业服务公司的特性导致的效率挑战
- **信息传递效率**：后续服务环节的部门经手后需要重新梳理材料
- **客户协同需求**：需要引导客户使用飞书协作，存在客户接受度问题

### 4.3 技术集成需求
- **多平台整合**：使用飞书前需要同时打开多个平台的工具软件
- **系统兼容性**：与现有业务系统的深度集成需求
- **数据迁移**：从分散平台到统一平台的数据迁移挑战

### 4.4 规模化扩展需求
- **人才复制**：随着业务扩展，如何快速复制专业人才
- **服务标准化**：如何在保证专业性的同时实现服务标准化
- **知识更新**：行业法规变化快，需要持续更新知识库

从多维表格管理数据，到文档搭建知识库，再到妙记开客户会，飞书产品线的全面应用产生了协同效应。
