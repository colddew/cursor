# 咿呀英博口腔集团飞书客户案例分析报告

## 1. 公司基本信息

### 企业概况
- **公司名称**: 咿呀英博口腔集团
- **成立时间**: 2003年
- **所属行业**: 医疗健康-口腔医疗服务
- **企业规模**:
  - 员工总数: 4000余名
  - 医疗团队: 1500余名
  - 门诊数量: 60余家连锁机构
  - 年服务客户: 超过180万客户

### 业务布局
- 全面布局口腔医疗服务全产业链
- 旗下拥有数家口腔品牌和加工厂
- 具备资深医疗团队和高素质人才梯队
- 多次获得国家认证、行业认可

## 2. 飞书应用场景

### 2.1 精细化管理系统
**应用产品**: 飞书多维表格
- **数据一键填报**: 员工选择填报月份，在预设模板中进行实时情况填报
- **权限规范管理**: 各战区、各门诊及个人数据逐级权限控制
- **流程自动推进**: 填报任务每日定时推送，分级管理与催办机制

### 2.2 AI客服系统
**应用产品**: 飞书服务台 + 飞书智能伙伴
- **统一入口**: 飞书服务台作为统一咨询入口
- **AI问答机器人**: 处理高频问题，复杂情况转人工
- **人机协同**: AI客服覆盖全部门店，解答率达70%

### 2.3 日报管理系统
**应用产品**: 飞书小程序 + 智能仪表盘
- **任务目标分发**: 各级任务目标层层分发、自动提醒
- **移动端填报**: 员工通过小程序入口填写日报
- **数据自动汇总**: 核心数据自动汇总、通晒
- **实时仪表盘**: 管理者实时查看全局进度和团队表现

## 3. 新获得的能力

### 3.1 数据管理能力
- **自动化报表生成**: 月度报表自动输出，节省3-4天统计时间
- **智能填报系统**: 每月节省填报数据工时2600小时
- **数据安全管控**: 多层级隐私权限设置，数据风险降低超过90%
- **实时数据反馈**: 数据疑问可实时反馈，跨部门沟通成本降低超过30%

### 3.2 协作效率提升
- **团队协作提效**: 整体协作效率提升40%
- **响应速度优化**: 问题响应从漫长等待到秒级即达
- **沟通成本降低**: 每日定点推送填报提醒，节约1-2小时人工通知

### 3.3 管理能力升级
- **目标管理精准化**: 目标清晰对齐，进度透明可视
- **考核自动化**: 考核规则自动落地执行，轻松实现日常监督
- **管理精准高效**: 管理时间从数小时压缩至分钟级
- **医疗质控提升**: 医疗质控能力提升30%

### 3.4 知识管理能力
- **知识沉淀**: 实现咨询响应、工单流转的全流程数字化
- **经验复制**: 一线员工满意度显著提升，服务能力实现质的飞跃
- **标准化管理**: 打造万店如一的"数字底座"

## 4. 未满足的需求

### 4.1 业务复杂度挑战
- **数据核对复杂性**: 日均核对超500条数据，月度信息量超过一万条
- **多环节匹配**: 涵盖牙位准确性、金额填报、医生业绩归属及外加工匹配等复杂环节
- **手工操作风险**: 庞大数据量叠加繁复手工流程，易出错、易遗漏

### 4.2 跨部门协作瓶颈
- **总部部门负担重**: HIS、行政、财务等核心部门业务繁重
- **海量门店咨询**: 应对海量门店咨询导致一线问题流转不畅
- **解答延迟**: 跨部门协作效率低下，问题解答延迟

### 4.3 知识管理压力
- **知识管理投入大**: 集团内部知识管理需投入大量精力
- **经验复制难**: 在规模化扩张中面临经验复制困难
- **一线支持获取难**: 各地员工高效获取总部支持成为挑战

### 4.4 目标管理痛点
- **目标传递走样**: 目标层层传递容易走样
- **进度追踪困难**: 进度卡在哪里谁也看不清
- **填表占用时间**: 主管手工汇总数据，员工时间被填表占据
