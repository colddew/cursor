# 迪尚集团飞书客户案例分析报告

## 1. 客户基本信息

**公司名称**：迪尚集团

**所属行业**：服装制造/纺织服装行业

**公司规模**：超5万名员工

**主营业务**：
- 跨国服装产销企业
- 服装贸易、国内工装以及品牌运营
- 为全球100多个国家的800多个服装品牌提供设计开发、生产、供应链和贸易服务
- 年均营收超150亿
- 全国最大服装出口及职业工装企业之一

**企业背景**：
- 创立于1993年
- 打造首个纺织服装垂直生态供应链平台
- 完成国庆70周年、建党100周年、北京冬奥会等重大活动服装任务
- 获"中国服装百强"等多项国家级荣誉资质

## 2. 飞书给客户带来的新能力和价值

### 2.1 具体使用的飞书功能模块

1. **即时消息与订阅号**
   - 即时消息功能触达全体员工
   - 11个订阅号覆盖文化-行政-资讯-业务全渠道
   - 实现消息精准推送和阅读状态监控

2. **知识管理模块**
   - 飞书知识库
   - 飞书云文档
   - 飞书服务台
   - 搭建超100个知识库

3. **协同工具**
   - 飞书多维表格
   - 流程管理功能
   - 串联40多个协同场景

### 2.2 解决的业务痛点

1. **信息传递不畅问题**
   - 解决领导任务派发与通知下达依赖口头传达的问题
   - 消除多种通信工具并用、信息分散难整合的挑战
   - 实现消息精准触达全员，提升信息沟通效率

2. **知识管理分散问题**
   - 整合分散在200多个业务部门的知识
   - 解决员工寻找文件流程复杂、效率低下问题
   - 降低员工离职带走知识和信息泄露风险

3. **跨部门协同难题**
   - 解决业务部门流程细碎易漏影响订单交付问题
   - 减少行政和财务部门手动操作，提高效率
   - 避免工装部门与外部供应商对接重复填写信息易出错

### 2.3 带来的业务价值和效率提升（量化数据）

1. **信息沟通效率提升**
   - 因信息传递不畅导致的工作延误减少了30%
   - 通过订阅号数据分析功能，实时掌握预览数据及访问数据，优化推送策略

2. **知识管理效率提升**
   - 信息获取效率提升8倍
   - 查询时间从1-2天缩短至10分钟速查
   - 简单信息询问从半小时缩短至10分钟
   - 提升客户响应速度和满意度

3. **业务协同效率提升**
   - 订单处理提效66%
   - 每季度为团队节约176小时的工作时间
   - 库存利用率提升30%
   - 一个人能够对接几百家门店
   - 耗时从30分钟缩短至10分钟
   - 在近4个月的订单操作中，对接500家门店，共节省约12.5天时间

### 2.4 具体应用场景

1. **企业文化传播**
   - 通过订阅号精准推送企业文化内容
   - 员工可根据需求选择关注频道

2. **医疗板块知识库**
   - 使用近150个云文档搭建知识库
   - 覆盖企业文化、组织结构、新人入职流程、产品手册等10个方面
   - 标准产品信息和价格录入知识库，销售可直接查询

3. **业务协作全流程系统**
   - 覆盖样品费管理、开发任务管理、大货生产周报、出运管理、发票付款等场景
   - 省区运营、销售门店及供应商三方协同平台

## 3. 飞书未能满足的客户需求

### 3.1 潜在未满足需求分析

从案例中可以看出，虽然飞书在迪尚集团的数字化转型中发挥了重要作用，但仍存在以下潜在未满足的需求：

1. **深度行业定制化需求**
   - 服装行业的特定工作流程可能需要更专业的定制化解决方案
   - 案例中提到了"从设计研发到生产制造的全链路数字化场景"，但未详细说明飞书在设计和生产制造环节的深度应用

2. **更高级的数据分析和AI应用**
   - 虽然提到了多维表格的数据分析功能，但未涉及更高级的预测分析或智能决策支持
   - 缺少对市场趋势、消费者行为等外部数据的整合分析能力

3. **供应链深度协同**
   - 案例中提到了与800多个品牌客户的合作，但未详细说明如何通过飞书实现与外部供应链伙伴的深度协同
   - 缺少对供应商管理、物流跟踪等供应链关键环节的具体解决方案

### 3.2 需要改进的功能或服务

1. **外部协作增强**
   - 虽然提到了与供应商对接，但可能需要更强的外部用户权限管理和协作功能
   - 可能需要更好的客户门户或供应商门户功能

2. **移动端优化**
   - 对于需要经常外出的销售人员和管理人员，可能需要更强的移动端功能
   - 特别是在工厂巡检、门店管理等场景下的移动应用

3. **行业特定模板**
   - 可能需要更多服装行业特定的业务模板和工作流程
   - 减少企业在自定义开发方面的工作量

### 3.3 客户可能需要的其他解决方案

1. **专业ERP系统集成**
   - 飞书可能需要与专业的服装行业ERP系统进行更深度集成
   - 实现从设计、生产到销售的全流程数据打通

2. **智能制造设备连接**
   - 作为服装智造企业，可能需要与智能制造设备的数据连接
   - 实现生产数据的实时监控和分析

3. **跨境电商支持**
   - 面向全球100多个国家的业务，可能需要更多的跨境电商相关功能
   - 多语言、多货币、多税务等国际化功能

## 4. 战略洞察

### 4.1 迪尚集团的数字化转型战略

迪尚集团显然在积极推进全面的数字化转型，从传统的服装制造企业向数字化、智能化企业转型。飞书作为协同办公平台，在其中扮演了"数字化基础设施"的重要角色。

### 4.2 飞书在制造业的价值定位

飞书在迪尚集团的成功案例表明，其在制造业的价值不仅局限于内部协同，更在于：
1. 打破部门壁垒，实现跨部门高效协同
2. 促进知识沉淀和传承，提升组织学习能力
3. 支持业务流程优化和效率提升
4. 为数字化转型提供统一的协作平台

### 4.3 未来发展方向

从迪尚集团的案例可以看出，制造业客户对数字化协作平台的需求正在从基础的沟通协同向更深层次的业务协同和智能决策发展。未来可能需要：
1. 更强的行业定制化能力
2. 更深度的业务系统集成
3. 更高级的数据分析和AI应用
4. 更完善的外部协作生态
