# 东风康明斯飞书客户案例分析报告

## 1. 客户基本信息

### 公司概况
- **公司名称**：东风康明斯发动机有限公司
- **所属行业**：制造业/汽车产业链
- **公司规模**：年标准产能23万台，300多名人员仅研发部门
- **主营业务**：生产多系列机械式及全电控柴油机，广泛应用于载重汽车、客车、工程机械、船用主辅机等领域
- **成立时间**：1996年
- **企业性质**：由东风汽车股份与美国康明斯合资成立
- **市场地位**：柴油中重卡国六市场份额行业第一

## 2. 飞书给客户带来的新能力和价值

### 2.1 使用的飞书功能模块

1. **飞书多维表格**：核心功能，用于项目管理全链路数字化
2. **仪表盘的AI智能分析功能**：数据分析和决策支持
3. **字段捷径功能**：包括总结功能和分类功能
4. **工作流引擎**：自动化流程管理
5. **系统连接能力**：与BPM、DPDM等系统集成

### 2.2 解决的业务痛点

#### 研发管理痛点
1. **资源不透明**：各职能工作计划和HC资源分布无法直观看到
2. **项目费用脱节**：项目状态、费用预测和实际费用单独管理，缺乏关联
3. **资源管理落后**：台架和工程车资源用线下表格管理，状态不可见

#### 采购协同痛点
1. **图纸管理混乱**：供应商提交图纸无统一管理，易出现版本混淆
2. **评审效率低下**：依赖人工比对和邮件往来，耗时长
3. **签署流程繁琐**：纸质版报价单需要跨厂区传递和线下签字

#### 安全/质量管理痛点
1. **信息滞后**：依赖Excel和邮件传递，数据更新不及时
2. **统计耗时**：手工统计效率低
3. **数据孤岛**：信息分散，难以统一分析

### 2.3 带来的业务价值和效率提升

#### 研发管理方面
1. **AI赋能**：
   - 智能分析：通过AI自动给出分析见解和改进建议
   - 自动打标：自动为项目等级进行标记
   - 公式生成：通过自然语言生成复杂计算规则

2. **数据整合**：
   - 连接BPM、DPDM等系统，自动获取3200多条工单需求和14500多条零件数据
   - 统一规范12类项目类型和337项开发任务标签

3. **资源可视化**：
   - 人员：统一13个科室模板，管理300多名人员的工作计划和工作天数资源
   - 设备：网上管理80多个台架和工程车资源，规范210多个费用标准
   - 财务：自动对比70多个项目的费用预测和实际花费

#### 采购协同方面
1. **图纸管理结构化**：建立统一数据表，自动归集供应商图纸
2. **智能评审**：利用字段捷径功能对评审意见进行分类，推动供应商相互对标
3. **电子签署**：自动生成报价单模板，通过工作流跟进催办

#### 安全/质量管理方面
1. **自动化流程**：自动流转隐患信息，无需人工干预
2. **智能化提醒**：分级提醒计划改进时间，超期自动通知部门主管
3. **实时监控**：动态仪表盘展示问题类别、完成率等核心指标

### 2.4 具体数据支撑

1. **直接经济效益**：
   - 直接降本超过100万元
   - 台架费用节约超过100万元
   - 人员管理效率提升价值超过10万元/年
   - 清牌赢分平台年节约人力成本50+万元

2. **运营效率提升**：
   - 清牌赢分平台月均节省500小时沟通时间
   - 整合需求降低开发成本，如铸三搬迁项目和C机一揽子国产化项目共同使用C8.3耐久台架
   - 提前发现无法按时完成的工单，降低延期风险

## 3. 飞书未能满足的客户需求

### 3.1 已识别的限制和挑战

1. **系统集成复杂性**：
   - 需要手动连接BPM、DPDM等现有系统
   - 需要自行制定数据规范和标准化流程

2. **AI功能限制**：
   - AI分析功能虽然提供了智能建议，但可能无法完全替代专业领域专家的深度分析
   - 自然语言生成公式的功能可能对复杂计算规则有一定局限性

3. **定制化需求**：
   - 需要针对特定业务场景（如台架和工程车管理）制定专门的费用标准和预测公式
   - 需要开发专门的工作流来满足特定业务流程需求

### 3.2 可能的改进空间

1. **行业特定模板**：
   - 缺乏针对制造业发动机研发的预制模板
   - 需要更多针对汽车产业链的专业化解决方案

2. **高级分析功能**：
   - 可能需要更深入的成本效益分析功能
   - 项目风险评估功能可能需要进一步强化

3. **移动端支持**：
   - 对于需要经常在现场的工程师和采购人员，可能需要更强的移动端支持

### 3.3 客户可能需要的其他解决方案

1. **专业项目管理工具集成**：
   - 可能需要与更专业的项目管理工具（如MS Project）集成
   - 可能需要与ERP系统更深度的集成

2. **专业供应链管理**：
   - 对于复杂的采购协同，可能需要更专业的SCM（供应链管理）系统
   - 可能需要更强大的供应商管理功能

3. **专业质量管理系统**：
   - 对于安全/质量管理，可能需要与专业的QMS（质量管理系统）集成
   - 可能需要更强大的合规管理功能

## 4. 战略洞察

### 4.1 飞书的核心优势

1. **高度灵活性**：多维表格提供了强大的自定义能力，能够适应复杂的业务场景
2. **AI驱动效率**：AI功能显著降低了数据分析和技术门槛
3. **集成能力**：能够连接现有系统，打破数据孤岛
4. **成本效益**：相比专业系统，提供了更高的性价比

### 4.2 制造业数字化转型趋势

1. **从经验驱动到数据驱动**：传统依赖经验的管理方式正被数据可视化和智能分析取代
2. **从孤岛到整合**：跨部门、跨系统的数据整合成为关键需求
3. **从手工到自动化**：重复性工作正被自动化流程替代

### 4.3 未来发展机会

1. **行业解决方案深化**：针对制造业特定场景提供更专业的解决方案
2. **AI能力扩展**：在特定领域提供更专业的AI分析能力
3. **生态系统建设**：与更多专业软件厂商建立合作关系

## 结论

东风康明斯的案例展示了飞书在制造业数字化转型中的强大能力，特别是在研发项目管理、采购协同和质量安全管理方面。通过多维表格和AI功能，飞书帮助东风康明斯实现了显著的效率提升和成本节约。然而，在某些专业领域，仍需要与其他专业系统配合使用，这也为飞书未来的产品发展提供了方向。
