# 非夕科技飞书客户案例深度分析报告

## 1. 客户基本信息

### 公司名称
非夕科技 (FLEXIV)

### 所属行业
智能制造/通用智能机器人制造业

### 公司规模
独角兽企业，全球技术领先的通用智能机器人公司

### 主营业务
- 研发、生产集工业级力控、计算机视觉和人工智能技术于一体的自适应机器人产品
- 为不同行业的客户提供基于非夕机器人平台型产品的创新性解决方案和服务
- 专注于机器人技术的革新，致力于为智能制造带来无限可能

### 公司背景
- 成立时间：2016年
- 核心创始团队：来自斯坦福大学机器人和人工智能实验室
- 发展历程：2022年成长为通用机器人领域的独角兽企业
- 技术特点：秉承仿人化创新理念，专注于自适应技术领域

## 2. 飞书给客户带来的新能力和价值

### 2.1 核心使用的飞书功能模块

1. **飞书多维表格** - 核心业务数据管理
   - 商机管理全流程一站式管理
   - 发货协同管理系统
   - 机器人异常信息统计表格
   - 质保数据分析

2. **飞书审批** - 业务流程管理
   - 与OA系统集成的项目流程管理
   - 审批流程自动化

3. **飞书文档** - 知识沉淀与共享
   - 信息收集与共享
   - 知识资产管理

4. **飞书会议室** - 协同沟通
   - 团队协作和会议管理

### 2.2 解决的业务痛点

#### 商机管理痛点
- **原有问题**：Salesforce系统账号成本高昂，75%员工无法实时访问业务进展
- **影响**：跨部门协作效率低下，信息流通不畅

#### 发货管理痛点
- **原有问题**：发货信息协同不及时，进度难以把控
- **历史尝试**：先使用Anyshare Excel表格，后尝试云文档，但均存在功能限制
- **影响**：沟通效率低下，隐形成本增加

#### 质保管理痛点
- **原有问题**：质保流程缓慢，人工依赖度高
- **原有方式**：依赖线下Excel表格记录，信息来源分散
- **影响**：沟通成本增加，工作效率降低，存在数据丢失和错误风险

### 2.3 带来的业务价值和效率提升

#### 商机管理价值
- **用户覆盖扩展**：从仅BD和Sales人员扩展到100多名员工（包括方案工程师、关键技术工程师、机器人调试工程师、产品经理、研发等多个岗位）
- **信息透明度提升**：实时更新和共享项目信息，确保信息流转效率
- **系统集成**：建立了与OA系统集成的项目流程，构建成本更低、功能更全面的新型商机管理系统

#### 发货管理价值
- **一体化协同管理**：打通业务团队与供应链之间的沟通壁垒
- **实时跟踪调整**：业务人员能够实时跟踪和调整发货计划
- **自动化同步**：通过自动流程迅速将调整同步至供应链部门
- **系统化管理体系**：建立透明、系统化的发货流程管理体系

#### 质保管理价值
- **自动化管理系统**：打造自动化质保管理系统，实现数据管理清晰化
- **风险预警机制**：自动化提醒功能加速返修信息同步，及时发现潜在质量问题
- **数据分析能力**：借助仪表盘进行定期质量审核和评估，持续推动产品优化

### 2.4 具体数据支撑

1. **IT成本节省**：
   - 商机管理IT成本每年减少80万元
   - 通过替代昂贵的Salesforce账号，实现显著成本节约

2. **效率提升数据**：
   - 发货管理协同效率提升50%
   - 质保数据分析效率提升67%
   - 月度数据统计时间从180分钟降至60分钟

3. **业务流程优化**：
   - 100多名员工可实时访问业务进展（原来只有25%员工有权限）
   - 实现了从商机到收款(LTC)全链路业务升级

## 3. 飞书未能满足的客户需求

### 3.1 已识别的局限性

1. **原有系统集成需求**：
   - 需要与OA系统集成以实现完整的项目流程管理
   - 表明飞书本身在复杂企业系统集成方面可能存在局限性

2. **专业功能限制**：
   - 在发货管理过程中，非夕经历了多轮改革尝试，说明可能需要更多专业化的物流管理功能
   - 质保管理仍需借助仪表盘等外部工具进行数据分析

3. **行业特定需求**：
   - 作为机器人制造企业，可能需要更多针对制造业的专业功能模块
   - 缺少针对智能制造行业的专门化解决方案

### 3.2 潜在的未满足需求

1. **高级数据分析能力**：
   - 虽然提升了数据分析效率，但仍需依赖外部工具进行深度分析
   - 可能需要更强大的AI驱动的预测分析和决策支持功能

2. **供应链深度集成**：
   - 作为制造企业，可能需要与更多供应链系统进行深度集成
   - 缺少针对制造业生态系统的专门集成解决方案

3. **客户服务管理**：
   - 案例主要聚焦内部流程优化，对于客户服务和售后管理的深度支持可能不足
   - 作为机器人公司，客户服务是关键业务环节

4. **研发项目管理**：
   - 作为技术公司，研发项目管理是核心业务，但案例中对此提及较少
   - 可能需要更专业的研发项目管理和协作工具

### 3.3 功能改进建议

1. **增强系统集成能力**：
   - 提供更丰富的预置集成接口
   - 支持更复杂的企业系统集成场景

2. **行业专业化**：
   - 开发针对制造业的专业化功能模块
   - 提供行业特定的模板和最佳实践

3. **高级分析功能**：
   - 内置更强大的数据分析和可视化能力
   - 集成AI驱动的预测分析功能

4. **扩展业务覆盖**：
   - 加强对客户服务管理的支持
   - 提供更全面的研发项目管理功能

## 4. 战略洞察

### 4.1 飞书的核心价值主张
1. **成本效益显著**：通过替代昂贵的专业系统，实现大幅成本节约
2. **协作效率提升**：打通部门壁垒，实现信息透明和实时共享
3. **灵活性和适应性**：能够适应不同行业的特定需求
4. **一体化解决方案**：提供从沟通到协作到管理的全方位支持

1. **全链路数字化**：从商机到质保的完整业务流程数字化
2. **协作模式革新**：打破传统部门壁垒，实现跨部门高效协作
3. **数据驱动决策**：通过数据收集和分析支持业务决策

3. **智能化升级**：融入更多AI功能，提供智能决策支持
4. **全球化支持**：支持跨国企业的全球化协作需求

## 5. 总结

非夕科技案例展示了飞书在支持制造业企业数字化转型方面的显著价值，特别是在成本控制、效率提升和协作优化方面。通过飞书多维表格等核心功能，非夕科技成功实现了商机管理、发货协同和质保管理的全面升级，创造了可量化的业务价值。

然而，案例也反映出在专业系统集成、行业特定功能和高级分析能力方面仍有提升空间。这为飞书未来在制造业市场的产品优化和发展方向提供了有价值的参考。

非夕科技的成功实践证明了飞书作为企业协同平台在支持复杂制造业业务流程方面的能力，同时也为其他制造企业的数字化转型提供了可借鉴的经验。
