# 去哪儿飞书客户案例分析报告

## 1. 客户基本信息

**公司名称**：去哪儿（Qunar）

**所属行业**：互联网/在线旅游服务

**公司规模**：5001-10,000人

**主营业务**：去哪儿是成立于2005年的综合旅行服务平台，长期以价格优势为特色，现已拥有超过两亿交易用户。平台核心用户集中于25至35岁，00后用户增长迅速，大学生群体覆盖率居行业前列。去哪儿已与全球逾100家航空公司和9000家旅行代理商建立合作，支持实时预订68万余条航线、320万家酒店、签证、度假、景区门票、火车票、租车及接送机等多项服务，覆盖全球多个目的地。

## 2. 飞书给客户带来的新能力和价值

### 2.1 具体使用的飞书功能模块

- **多维表格**：核心数据存储和处理平台
- **工作流**：自动化流程处理
- **AI大模型**：集成DeepSeek、qmoss等工具实现智能分析
- **仪表盘**：数据可视化展示
- **消息推送**：自动通知与结果展示
- **OnePage项目模板**：项目协作管理
- **开放API**：与自研BI系统集成

### 2.2 解决的业务痛点

1. **用户问题分析效率低下**：
   - 原本各渠道系统相对独立，数据分散
   - 问题排查过程严重依赖人工
   - 以国内机票业务为例，每日收集问题约1460个，但仅能完成约35个问题的有效归因，分析覆盖率不足2%

2. **数据使用体验差**：
   - 更新滞后、移动端适配差
   - 报表依赖人工推送
   - 影响业务团队的响应效率和分析体验

3. **项目管理协同效率低**：
   - 并行百余项目、覆盖2000+产研人员的高频协作需求
   - 信息分散、节点不透明
   - 碎片化提醒方式效率低下

### 2.3 带来的业务价值和效率提升

1. **用户问题分析自动化体系**：
   - 构建了"多维表格+工作流+AI大模型"的自动归因体系
   - URS问题分析率从2%提升至98%，实现几乎全量问题的高效归因
   - 分析时间从原本一个问题需要两小时缩短至不到一分钟
   - 整体人效日均提升达96人天

2. **数据协同与看数效率提升**：
   - 技术运营中心打通自研BI系统与多维表格
   - 支持十余业务线看数需求
   - 整体使用效率提升2–6倍
   - 综合估算节省潜在开发成本50人日
   - 业务同学无需等待排期开发，平均1天内即可自助配置报表

3. **项目管理自动化**：
   - PMO团队通过自动化工具优化需求评审会及多项目协同流程
   - 借助多维表格串联会议预约、日程安排与信息同步
   - 实现RDQA需求评审全流程自动化
   - 每月减少约44人日的人力投入
   - 构建"事找人"机制、沉淀协作经验

### 2.4 具体数据支撑

- **98%**：URS问题归因率从2%提升至98%
- **2–6倍**：提升业务看数与报表效率2–6倍
- **44人日**：需求评审会议自动化每月节省44人日人力
- **96人天**：整体人效日均提升
- **50人日**：预计节省系统工时

## 3. 飞书未能满足的客户需求

根据案例分析，飞书主要在以下方面可能还存在改进空间或未能完全满足的需求：

1. **系统集成复杂性**：
   - 需要额外集成DeepSeek、qmoss等第三方工具才能实现完整的AI分析流程
   - 表明飞书自身的AI能力在特定业务场景下可能还不够完善

2. **移动端体验**：
   - 案例中提到原有系统存在"移动端适配差"的问题
   - 虽然飞书多维表格解决了部分问题，但可能移动端的完整体验仍有提升空间

3. **行业特定解决方案**：
   - 旅游业特有的复杂业务流程（如机票预订、酒店管理等）可能需要更多行业定制化功能
   - 案例中显示需要大量自定义开发才能满足特定业务需求

4. **大规模数据处理能力**：
   - 面对每日1460个问题的大规模数据处理需求
   - 可能需要更强的数据处理和分析能力

## 4. 战略洞察

1. **飞书的核心价值**：作为平台级产品，飞书通过多维表格、工作流等基础能力，结合开放API，为企业提供了灵活的数字化转型基础架构，而非单一的解决方案。

2. **AI赋能的重要性**：去哪儿案例凸显了AI大模型与业务流程结合的巨大价值，实现了从"靠人力"向"靠系统"的结构性转变。

3. **行业数字化趋势**：即使是互联网公司如去哪儿，也面临着系统孤岛、数据分散、人工依赖等数字化挑战，飞书提供了有效的整合解决方案。

4. **人效提升显著**：通过自动化工具替代人工操作，在问题分析、数据使用、项目管理等场景均实现了显著的人效提升，证明了数字化转型的投资回报。

这个案例充分展示了飞书作为企业协作平台的价值，特别是在AI赋能、自动化流程和数据协同方面的能力，同时也揭示了在特定行业场景下，飞书需要与其他专业工具结合才能发挥最大效用。
