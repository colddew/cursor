# 少年得到 - 飞书客户案例分析报告

## 📊 客户基本信息

- **公司名称**: 少年得到
- **所属行业**: 互联网教育
- **公司规模**: 中等规模，具备完整的产研中心和管理团队
- **主营业务**: 青少年儿童底层能力培养和综合素养提升的互联网教育
- **成立时间**: 2018年
- **目标用户**: 5-15岁年龄段青少年儿童
- **产品线**:
  - 《泉灵的表达素养课》- 培养学习力、思辨力、情商力、创造力
  - 《名著领读计划》- 培养自主阅读习惯
  - 综合类素养音视频课程
  - 特色性和个性化营地及创新类业务
- **企业使命**: 成为中国家庭素养教育的首选公司
- **企业愿景**: "让成长看得见"

## 🚀 飞书给客户的新能力/赋能

### 1. 高效会议协作能力

#### 飞阅会模式创新
- **会议效率提升30%**: 通过基于飞书文档的阅读式会议模式，彻底改变传统会议形式
- **会前预读机制**: 参会者限时阅读、批注材料，确保充分理解会议内容
- **会议时间大幅缩短**: 产研团队一半以上的会议不开，开的会议平均10分钟结束
- **高管例会优化**: 高管经营例会从长达3小时大幅缩短，提升决策效率

#### 脱PPT化协作
- **文档为核心**: 所有内容通过文档协作，摒弃传统PPT束缚
- **实时共享讨论**: 会议中实时共享想法，既保证灵活性又提升效率
- **文化层面变革**: 从"向我汇报"转向"context而非control"，促进主动参与

### 2. 知识管理体系建设

#### 产研公约落地
- **文档渗透率达90%**: 通过飞书建立完整知识库，实现高效知识共享
- **标准化流程**: 促进产研流程标准化，建立完整的内容开发体系
- **多人实时编辑**: 解决之前文档散落微盘、无法协同编辑的问题

#### 培训体系升级
- **"产品设计及定位"培训**: 2023年3月发起的公司级战略项目
- **方法论沉淀**: 通过飞书文档编制培训教材，发布研讨工具和任务
- **八大核心工具**: 最终形成产品设计及定位的完整工具体系
- **实战营结合**: 培训与实战营结合，通过实战演练不断打磨完善

### 3. 战略管理与人才盘点数字化

#### "一页纸"管理模板
- **项目一页纸**: 统一模板管理战略项目，简化汇报流程
- **预对齐机制**: 会前各业务单元深入研讨，提升会议实质性
- **亮点与问题识别**: 系统性提取项目亮点、识别问题、共创解决方案

#### 人才管理专业化
- **九宫格工具**: 专业的关键人才盘点和复盘工具
- **组织架构图**: 可视化人才结构分析
- **多维表格**: 高效的人才信息管理
- **精细权限设计**: 实现信息分级管理和保护
- **任务跟进**: 日程提醒和任务指派确保人才发展任务及时执行

### 4. 组织文化建设

#### 协作共享文化
- **全员参与模式**: 高效互动的会议模式促进员工主动参与
- **Context而非Control**: 强调提供背景信息而非简单控制
- **成长导向**: 以"让成长看得见"为宗旨的企业文化落地

### 5. 具体使用功能模块
- **飞书文档**: "飞阅会"协作、产研公约、培训教材、管理模板
- **飞书视频会议**: 高效会议讨论支持
- **飞书多维表格**: 用户旅程记录、人才数据管理
- **飞书知识管理**: 完整知识库体系建设
- **飞书绩效**: 人才盘点流程支持
- **飞书会议室**: 高效会议环境

## ❌ 飞书没有满足客户需求的部分

### 1. 会议文化转型的组织挑战
- **共识建立困难**: 飞阅会需要整个团队有共识才能推动，说明存在组织变革阻力
- **传统会议惯性**: 虽然效率提升，但仍然需要面对传统会议文化的惯性
- **文化变革成本**: 需要董事长亲自带队才能推行，说明推广成本较高

### 2. 人才培养体系的局限性
- **培训成本投入**: 需要投入大量资源进行"产品设计及定位"培训和实战营
- **经验复制难度**: 虽然建立了知识库，但业务骨干的培养仍需要时间沉淀
- **跨部门协作**: 虽然文档渗透率高，但跨部门协作的实际效果需要持续验证

### 3. 战略执行的可视化挑战
- **项目追踪深度**: 虽然有"一页纸"管理，但复杂项目的深度追踪仍有挑战
- **数据整合需求**: 人才盘点和项目管理的数据整合可能需要更多工具支持
- **决策支持系统**: 战略决策的数据化支持可能还需要进一步增强

### 4. 技术与业务的深度结合
- **产品开发效率**: 虽然产研协同提升，但产品开发效率的量化指标不够明确
- **用户反馈闭环**: 缺乏用户反馈到产品开发的完整闭环机制描述
- **数据驱动决策**: 业务数据与决策过程的结合还需要更深入的探索

### 5. 教育行业特殊需求
- **教学内容管理**: 缺乏针对教育行业特有的教学内容管理功能
- **学习效果追踪**: 没有学生学习效果追踪和分析的专业工具
- **家校沟通平台**: 缺乏与家长沟通的专用渠道和工具
- **AI教育应用**: 未提及AI功能在教育内容创作和个性化推荐方面的应用
