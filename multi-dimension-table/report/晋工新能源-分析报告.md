# 晋工新能源飞书客户案例分析报告

## 1. 客户基本信息

### 公司概况
- **公司名称**: 福建晋工新能源科技有限公司
- **所属行业**: 新能源工程机械制造业
- **公司规模**: 初创型公司（未明确员工数量，但根据业务性质应为中小型规模）
- **主营业务**: 
  - 专注于新能源领域的工程机械和矿用车等设备
  - 提供在钢铁、矿山、港口、基建和隧道等场景下的电动化、智能化和无人化装备系统解决方案
  - 2023年接受国电投旗下融和科技投资，加入国电投新能源生态链布局，成为"风光储充运"产业链的重要一环

## 2. 飞书给客户带来的新能力和价值

### 2.1 使用的飞书功能模块

1. **飞书OKR** - 目标管理与战略执行
2. **飞书多维表格** - 数据管理与跨部门协作
3. **飞书知识库** - 知识沉淀与信息共享
4. **飞书文档** - 协作文档与流程管理
5. **飞书招聘** - 人才招聘管理
6. **飞书绩效** - 员工绩效管理
7. **飞书会议室** - 会议管理与协作

### 2.2 解决的业务痛点

1. **目标管理挑战**
   - 痛点：新能源工程机械客户群体均为大中型企业，决策周期长，获客成本高，团队需要信息同步并快速协同
   - 解决方案：通过OKR将目标拆解成每月的销售台量和团队达成率等关键步骤，让员工更清晰地了解工作目标和重点

2. **数据孤岛问题**
   - 痛点：各业务部门的"数据中心"独立存在，跨部门协作时需要人工口头、群聊或会议形式传达数据
   - 解决方案：通过多维表格构成工程机械设备数据中台，实现设备从商机阶段、订单阶段到交付阶段及设备调度的全生命周期管理

3. **知识管理混乱**
   - 痛点：资料分散，未形成公司层级的知识库，新员工获取信息不友好
   - 解决方案：使用飞书知识库统一管理各部门资料和知识，包括公司规章制度、产品资料、营销工具包、行业资讯等

### 2.3 带来的业务价值和效率提升

1. **目标管理效率提升**
   - 目标完成率提升 **30%**
   - OKR填写率达到 **95%** 以上
   - 通过OKR激发员工主动思考与完成目标的积极性，形成勇于试错、敢于挑战的习惯
   - 销售业绩对比KPI机制有成倍提升

2. **数据管理效率提升**
   - 建立了统一的"数据中台"，1张多维表格管理全业务流程
   - 实现设备全生命周期管理，赋予每台设备一份"档案"
   - 各区域经理可通过多维表格表单填报商机信息
   - 业务人员可实时查看库存、设备型号、状态及地区分布等信息
   - 可实时跟踪设备使用状态、时间和轨迹

3. **知识管理效率提升**
   - 资料获取方式更加方便和统一
   - 知识库承载行业特色实践，帮助员工更好地了解不同工况的设备、销售方式或解决方案
   - 针对不同施工情况提供不同的场景化服务方案
   - 技术、营销、品牌各部门分别维护负责的内容，形成完整的业务方案

### 2.4 具体数据支撑

- **目标完成率提升30%**
- **OKR填写率95%**
- **销售业绩成倍提升**（相对KPI机制）
