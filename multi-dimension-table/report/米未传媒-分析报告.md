# 米未传媒飞书客户案例分析报告

## 1. 客户基本信息

**公司名称**：米未传媒

**所属行业**：文娱传媒/互联网内容创作

**公司规模**：未明确提及具体人数，但从案例描述可推断为中等规模的内容创作公司

**主营业务**：创作优质的互联网内容产品，致力于让世界更快乐。主要产品包括《乐队的夏天》《奇葩说》《一年一度喜剧大赛》等引爆网络的综艺节目

## 2. 飞书给客户带来的新能力和价值

### 2.1 具体使用的飞书功能模块

- **飞书文档**：用于协同编辑和侧边栏实时评论
- **飞书妙记**：自动记录讨论内容，分辨不同说话人
- **飞书多维表格**：演员招募流程管理、客户关系管理
- **飞书工作台**：统一的工作平台
- **飞书日历**：时间管理和日程安排
- **飞书绩效**：绩效管理
- **飞书知识库**：知识沉淀和管理

### 2.2 解决的业务痛点

**痛点1：远程办公的情绪连接问题**
- **挑战**：作为内容团队，米未很在意团队讨论选题时的情绪传递，但由于疫情开始居家远程办公，难以保持情绪连接
- **解决方案**：使用飞书文档协同编辑，侧边栏实时评论，能够实时记录每个人在每个时间点的不同反应，在线上建立独特的情绪线

**痛点2：演员招募流程繁琐**
- **挑战**：节目筹办期间需要处理大量演员报名、面试信息，此前使用多种工具导致流程繁琐，效率低下
- **解决方案**：使用飞书多维表格，整个面试流程都可在一张表格中流畅地完成

**痛点3：客户资源管理混乱**
- **挑战**：艺人经纪组的商务资源没有统一整合，公司资产无沉淀，存在客户资源流失风险
- **解决方案**：使用飞书多维表格建立「客户关系管理表」，统一管理公司客户资源

### 2.3 带来的业务价值和效率提升

**量化收益**：
- **5→1**：5个工具完成的招募流程，现在跑在一个飞书上
- **4小时**：节目组每天可节约面试信息处理时间
- **5小时**：艺人经纪组每周节省的周会时间

**质性收益**：
- **情绪连接**：在线上建立了独特的情绪线，保持远程团队的情绪传递
- **流程简化**：原本需要5个以上工具完成的演员招募流程，现在在一张多维表格中完成
- **数据沉淀**：自动记录讨论内容，无需人力手动记录
- **风险降低**：极大地降低了客户资源流失的风险
- **协同提升**：飞书的使用分享成为米未每周例会的常规内容

### 2.4 具体应用场景

**导演组应用场景**：
- 使用飞书文档开选题会，实时沉淀一切沟通细节
- 使用飞书妙记记录讨论过程，自动分辨不同说话人，自动换行
- 通过侧边栏实时评论保持情绪连接

**节目组应用场景**：
- 使用多维表格管理演员招募流程
- 候选人在多维表格报名表单中报名，数据自动同步
- 直接在表格中预览作品附件，无需下载
- 评委评分自动排序，随时通过飞书搜索找到候选人信息

**艺人经纪组应用场景**：
- 建立客户关系管理表，统一管理客户资源
- 自动分配可见权限给leader
- 系统自动发送飞书消息同步客户进展
- 通过双向关联字段防止撞单和客户分配不合理
