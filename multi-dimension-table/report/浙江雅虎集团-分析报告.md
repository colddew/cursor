# 浙江雅虎集团飞书客户案例分析报告

## 1. 客户基本信息

### 公司名称
浙江雅虎汽车部件股份有限公司（简称"浙江雅虎"）

### 所属行业
汽车零部件制造行业

### 公司规模
根据案例信息，公司在上海、成都、长春、天津、安徽等地设有子公司，并设有佛山、长沙、青岛等办事处网点，属于大中型企业

### 主营业务
主要从事汽车座椅骨架总成及零配件，以及相关模具的开发、生产和销售。本部坐落在中国汽摩配之都——浙江瑞安。

## 2. 飞书给客户带来的新能力和价值

### 2.1 具体使用的飞书功能模块

#### 项目管理场景
- **多维表格**：用于APQP计划管理、项目台账管理、项目总览表及可视化看板
- **飞书集成邮箱、视频会议和即时通讯工具**：保障沟通渠道畅通
- **项目审批流程**：实现规范化、线上化与自动化审批

#### 质量管理场景
- **质量数据化监控平台**：实时数据监测与分析系统
- **知识管理系统**：质量管理过程文件和经验教训留存与查阅
- **自动化数据处理和报表生成**：取代手工制表工作

#### 人力资源管理场景
- **飞书招聘**：招聘流程管理与候选人信息管理
- **飞书人事**：员工全生命周期管理
- **自动化审批流程**：人事流程审批线上化
- **人员报表系统**：按需快速搭建不同类型的人员报表

### 2.2 解决的业务痛点

#### 项目管理痛点
- **客户需求传递失真**：传统沟通方式导致需求易失真、遗漏
- **外部协作困难**：与供应商缺乏统一协作渠道，信息传递不及时准确
- **客户维护成本高**：缺少实时更新的项目进度展示平台
- **人力配置不均**：人力资源分配不合理，团队协作效率低

#### 质量管理痛点
- **问题处理速度慢**：质量处理进度追踪依赖线下纸质记录，易出现记录误差
- **质量问题追溯追责难**：质量数据分散于多个孤立业务系统
- **知识经验沉淀不足**：缺乏统一的知识沉淀平台，经验难以结构化流传

#### 人力资源管理痛点
- **招聘效率低**：第三方渠道招聘涉及网站多、操作繁琐，沟通耗时长
- **人事资料归档难**：员工材料签批、整理归档耗时且易出错
- **团队情况分析难**：员工信息分散管理，查看团队情况耗时长

### 2.3 带来的业务价值和效率提升

#### 具体数据支撑
- **项目管理效率提升60%**：通过数字化手段实现项目全链高效执行
- **质量管理每年节省300人天**：自动化数据处理和报表生成取代手工工作
- **员工入转调离流程100%线上化**：实现无纸化办公
- **每月为招聘HR节省40小时**：优化招聘流程
- **每月为人事HR节省61.12小时**：人事管理自动化
- **85%以上管理者**：每月通过飞书人事查看团队及人员信息

#### 具体价值体现

**项目管理价值**
1. **提速响应**：缩短从客户需求获取至试产环节的速度，更迅速完成试产并投入量产
2. **精准洞察**：构建高效客户沟通机制，管理者可迅速精准定位相关内容，及时提供专业回应
3. **人力优化**：实现人力资源科学配置，确保员工在合适岗位发挥价值

**质量管理价值**
1. **质量闭环管理升级**：实时监测质量参数，异常迅速捕捉，启动快速反馈机制
2. **跨企业沟通优化**：实时掌握质量问题及处理情况，反馈给下游客户，提高客户满意度
3. **知识沉淀线上化**：质量管理过程文件和经验留存，便于随时查阅，传承经验
4. **解放人力**：自动化处理取代繁琐手工工作，员工有更多时间投入深度思考

**人力资源管理价值**
1. **招聘流程优化**：HR、面试官与候选人高效信息交互，缩短沟通时间与流程周期
2. **人事管理自动化**：员工全生命周期线上化管理，降低人工工作量与错误率
3. **团队管理数据化**：HR按需快速搭建人员报表，为管理者提供团队洞察

## 3. 飞书未能满足的客户需求

- **生产设备数据对接**：汽车零部件制造业需要与生产设备数据实时对接，飞书在这方面可能需要加强
- **供应链协同平台**：虽然提到了与供应商的沟通改善，但更深层次的供应链协同功能可能不足
