# 珀莱雅飞书客户案例分析报告

## 1. 公司基本信息

**公司名称：** 珀莱雅化妆品股份有限公司

**行业：** 美妆护肤行业

**规模：** 国货美妆上市龙头企业

**主营业务：**
- 旗下拥有"珀莱雅""彩棠""Off&Relax""悦芙媞""CORRECTORS""INSBAHA原色波塔""优资莱""韩雅"等品牌
- 覆盖大众精致护肤、彩妆、洗护、高功效护肤等美妆领域
- 2023年以89.05亿元营收登顶国货美妆第一

## 2. 飞书应用场景

### 2.1 新品洞察分析
- **使用产品：** 飞书知识库、飞书多维表格、飞书订阅号
- **应用描述：** 从二维化文档到多维化系统，实现新品洞察3.0时代
- **具体功能：**
  - 新品观察看板：根据品类、关键词筛选市场新品，AI智能提炼卖点、成分，生成词云
  - 消费者画像看板：品牌间用户画像对比分析（性别、年龄、城市、消费力等）
  - 订阅号辐射全员：分享最新行业资讯至全公司

### 2.2 产品开发管理
- **使用产品：** 飞书多维表格
- **应用描述：** 一张表掌控新品开发全局，让项目管理更轻盈高效
- **具体功能：**
  - 承载提案、研发到上市的产品企划全流程
  - 实时查看从立项到产品最终上市的排期进度
  - 任务拆解下达、过程跟踪及全周期管理
  - 延迟节点特别追踪、异常提醒

### 2.3 直播样品管理
- **使用产品：** 飞书多维表格
- **应用描述：** 0成本建立直播样品管理系统，为高效开播、精准种草保驾护航
- **具体功能：**
  - 样品提需、领用、归还和盘点四大流程自动化记录
  - 样品库存实时更新，库存临界值实时预警
  - 高级权限管控，点对点对接领用人和样品管理员

## 3. 新获得的能力

### 3.1 数据分析能力升级
- **从碎片化到系统化：** 实现从1-1表格分享到1-全员的数据辐射
- **AI赋能分析：** 一键提炼新品卖点、成分，生成词云进行趋势洞察
- **多维度对比：** 行业关键品牌用户画像自助筛选对比
- **成本效益：** 0成本完成消费者数据资产的有效沉淀

### 3.2 项目管理能力提升
- **跨部门协同：** 40+位上海研发中心同事实时查看项目进度
- **效率提升：** 每天在核对排期、任务、文档更新工作中节省25%时间
- **风险管控：** 延迟节点特别追踪、异常提醒，关键环节资料上传
- **全周期管理：** 从立项到产品上市的全流程管理

### 3.3 样品管理数字化
- **流程自动化：** 样品出入库自动化记录，减少手工操作误差
- **实时库存管理：** 库存准确率达100%，对接及时率达100%
- **效率提升：** 工作时间节省30h/月，使用体验满意度95%+
- **预警机制：** 库存临界值实时预警，提前备货

### 3.4 信息传播能力增强
- **全员覆盖：** 订阅号将最新行业资讯分享至全公司
- **时效性提升：** 解决手动群发覆盖率低、滞后性强的问题
- **权限管理：** 知识库权限管理，确保机密数据不外泄

## 4. 未满足的需求

### 4.1 数据深度挖掘需求
- **数据激活不足：** 案例中提到"大量的数据没有做激活，利用率低，挖掘深度不够"
- **分析工具局限：** "公司数据庞杂，缺少高效易用的分析工具"
- **文档碎片化：** "文档形式的报告依然偏碎片化，数据资产还缺乏有效管理"

### 4.2 跨部门沟通效率
- **信息碎片化：** "跨部门沟通信息碎片化，无法及时追踪任务的进展"
- **缺乏实时看板：** "缺乏实时看板和风险预警"
- **多渠道管理：** "提案的监督、需求的反馈、进度的复盘等均需通过多渠道管理"

### 4.3 样品管理流程优化
- **手工登记统计：** 原先"样品的出入库都是手工登记统计的，导致实时库存不准确"
- **缺乏统一管理：** "缺乏统一和固定的管理模式以及完整可追溯的机制"
- **临时补样问题：** "无法提前备货，出现了比较多临时补样的情况，增加了发货成本"

### 4.4 行业资讯传播效率
- **传播覆盖率低：** "以往手动群发行业资讯、前沿信息传播覆盖率低"
- **信息滞后性：** "滞后性强的问题"
- **单点询问耗时：** "减少单点询问带来的时间损耗"
