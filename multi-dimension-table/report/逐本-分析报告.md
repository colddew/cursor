# 逐本飞书客户案例分析报告

## 一、公司基本信息

### 1.1 公司概况
- **公司名称**: 逐本
- **成立时间**: 2016年
- **行业**: 美妆护肤行业
- **品牌理念**: "追逐本质 · 追寻本心"
- **业务模式**: 自研＋自控供应链的创研道路

### 1.2 主营业务
- **产品线**: 卸妆品类和护肤品类
- **产品特色**:
  - 融合自然植愈成分
  - 现代前沿的护肤科技
  - 天然的芳香精油能量
- **市场认可**: 在卸妆品类和护肤品类中获得多方认可

### 1.3 企业规模与特点
- 新兴消费品牌，注重效率和成本控制
- 面临激烈的市场竞争和消费者偏好快速变化的挑战
- 积极推进数字化转型以应对市场变化

## 二、飞书应用场景

### 2.1 场景一：一站式审批平台
**使用产品**: 飞书审批 + 飞书多维表格

**应用详情**:
- 构建集中的审批平台，实现费用审批数字化升级
- 所有审批数据即时汇总并分类存储
- 实现统一的审批入口，便于实时管控费用成本
- 审批明细自动存档知识库，便于追溯和沉淀

**实施效果**:
- 每月节约财务部门3天的统计时间
- 无需二次拆解交通费、住宿费和餐补等明细项
- 审批明细同步时间成本和费用成本大幅降低

### 2.2 场景二：动态销售预测系统
**使用产品**: 飞书多维表格 + 飞书视频会议 + 飞书妙记

**应用详情**:
- 统一各渠道销售预测收集模板
- 保留各渠道原有的填写习惯，降低适应成本
- 自动收集各渠道提报信息，减少人工催收
- 建立销售预测提报群组，实现快速调整和实时同步

**实施效果**:
- 供应链响应速度提升20%
- 有效降低缺货和滞销风险
- 提升整体运营效率
- 支持数据导出和存档，解决版本管理问题

### 2.3 场景三：客服舆情管理系统
**使用产品**: 飞书多维表格 + 飞书智能伙伴 + 飞书文档

**应用详情**:
- 搭建舆情管理系统，记录和追踪客服处理动作
- 飞书智能伙伴进行深度舆情分析
- 实现舆情结果自动化预警
- 部门负责人实时更新处理进展，信息自动同步

**实施效果**:
- 舆情风险100%及时传递和解决
- 节省运营人员每周一天的工作量
- 投入产出比大幅提升
- 给下一步运营动作提供有价值的参考

## 三、新获得的能力

### 3.1 数字化流程管理能力
- **统一审批入口**: 实现所有审批流程的数字化管理
- **实时费用监控**: 财务团队能够实时查看和管理各类费用
- **自动化数据汇总**: 消除人工数据处理的延迟和错误
- **知识库自动沉淀**: 审批记录自动存档，便于追溯和分析

### 3.2 供应链协同能力
- **动态销售预测**: 实现月度/双周的滚动销售预测
- **跨部门信息同步**: 销售端与供应链数据实时对齐
- **敏捷响应机制**: 商品和数量调整可以快速完成
- **风险管控能力**: 有效降低缺货和滞销风险

### 3.3 舆情管理智能化能力
- **数字化舆情追踪**: 所有客服处理动作可被记录和追踪
- **AI驱动分析**: 飞书智能伙伴进行深度舆情分析
- **自动化预警机制**: 舆情风险自动推送至相关部门
- **透明化处理流程**: 舆情处理进度实时可见

### 3.4 数据驱动决策能力
- **实时数据洞察**: 获得业务实时数据支持决策
- **数据规范化**: 统一模板确保数据一致性和质量
- **历史数据分析**: 支持数据导出和存档，便于历史分析
- **跨部门数据共享**: 打破部门间数据孤岛

## 四、未满足的需求

### 4.1 系统集成局限
- **第三方舆情依赖**: 仍需第三方负责舆情抓取，信息时效性受影响
- **供应商分析不足**: 供应商提供的舆情分析内容过于简单，缺乏业务参考价值
- **手动分析需求**: 仍需要客服手动登记与逐一分析，工作效率有待提升

### 4.2 工作流程缺口
- **沟通链路冗长**: 在某些场景下仍存在沟通链路过长的问题
- **协作效率有待提升**: 虽然已有改善，但跨部门协作仍有优化空间
- **数据时效性挑战**: 某些关键数据的实时性仍有提升空间

### 4.3 分析能力限制
- **深度分析需求**: 对于复杂的业务分析，现有工具可能不够深入
- **预测精度提升**: 销售预测的准确性仍有提升空间
- **风险预警机制**: 可能需要更智能化的风险预警和应对机制

### 4.4 行业特殊需求
- **美妆行业特定需求**: 对于产品生命周期管理、季节性需求预测等行业特色需求，可能需要更定制化的解决方案
- **消费者洞察深度**: 对于消费者行为分析的深度和广度可能有更高要求
- **供应链复杂性**: 随着业务扩展，供应链管理复杂度增加，需要更强大的支持

随着业务的发展和市场的变化，逐本在数字化方面仍有提升空间，特别是在更深入的数据分析、更智能的预测能力和更完善的系统集成方面。但已有的数字化基础为未来的持续改进和扩展奠定了良好基础。
