# 峰潮信息飞书客户案例深度分析报告

## 1. 公司基本信息

### 1.1 公司概况
- **公司名称**: 成都峰潮信息技术有限公司
- **成立时间**: 2015年
- **行业性质**: 高水平专业数字化IT服务公司
- **主营业务**:
  - 计算机信息系统集成和运行维护
  - 计算机软硬件开发及技术咨询
- **业务覆盖范围**:
  - 行业领域：广电、军工、政企、交通、国电等行业
  - 地理范围：全国34个省级行政区，以及欧洲、北美、非洲等地区
- **企业定位**: 基于自研数字平台进行企业数字化管理和业务智能化运营的先锋

### 1.2 企业特点
- 项目制运营模式为主
- 人员规模：具有跨国业务布局
- 技术驱动型企业，自身也在进行数字化转型
- 业务具有明显的项目倾向性特征

## 2. 飞书应用场景

### 2.1 项目管理场景
- **应用产品**: 飞书项目、飞书文档
- **应用范围**: 2个事业部的20+条标准流程
- **主要功能**:
  - 不同项目类型自动带出流程节点
  - 项目更新自动通知相关人员
  - 人力排期直观展示
  - 质量管理自动化流程

### 2.2 合同管理场景
- **应用产品**: 飞书多维表格
- **主要功能**:
  - 项目合同数据统一汇总
  - 细化角色和部门权限管控
  - 权限精确到部门负责人、二级业务负责人、项目经理
  - 自动化流程触发（如回款节点提醒）

### 2.3 招聘管理场景
- **应用产品**: 飞书招聘、飞书日程
- **主要功能**:
  - 筛选、面试、面评及录用审批一站式操作
  - 招聘进度全透明
  - 全程线上留痕
  - 面试日程自动协调和提醒

### 2.4 整体办公协同
- **全员100%同频协作**
- 实现办公协同、人才招募、薪酬绩效、人事流转、项目进展的一站式管理

## 3. 新获得的能力

### 3.1 流程数字化能力
- **业务流程标准化**: 将20+条分散的业务流程统一沉淀到数字化平台
- **自动化流程管理**:
  - 项目创建时自动带出质量管理信息
  - 自动提醒相关人填写、登记、打分
  - 回款节点自动触发提醒流程
- **跨系统集成**: 打通办公软件与项目管理软件的数据壁垒

### 3.2 数据管理能力
- **统一数据平台**: 将分散的Excel表格整合为统一的多维表格
- **精细化权限控制**: 实现到个人级别的数据访问权限管理
- **实时数据同步**: 多人协作维护，数据自动汇总至主表
- **数据可视化**: 人力排期视图直观展示员工工作安排

### 3.3 协同办公能力
- **实时信息互通**: 项目更新自动通知到人
- **跨角色协作**: HR、面试官、外部联系人、候选人等多角色一站式操作
- **全流程留痕**: 招聘过程全程可追溯
- **智能调度**: 日程功能自动协调和提醒

### 3.4 管理决策能力
- **资源可视化管理**: 管理者通过人员视图直接看到员工空闲时间
- **质量管控自动化**: 六七十个项目并行时质量管理自动化
- **合同全生命周期管理**: 从立项到结项的完整数据追踪

## 4. 未满足的需求

- **原有系统割裂**: 招聘软件、沟通软件和面试软件不互通

- **数据收集困难**: 多项目并行时资料收集需要反复催促
- **权限管理复杂**: 不同部门数据保密需求难以满足

- **知识管理体系**: 作为技术服务公司，知识沉淀和复用可能是重要需求
- **移动端能力**: 案例中没有明确提到移动办公需求

### 4.3 行业特定需求
