# 汇川技术飞书客户案例分析报告

## 1. 客户基本信息

### 公司概况
- **公司名称**: 深圳汇川技术股份有限公司（简称"汇川技术"）
- **所属行业**: 工业自动化、制造业
- **公司规模**:

  - 分子公司: 30余家

  - 2022年营业利润: 43.20亿元

汇川技术专注于工业自动化控制产品的研发、生产和销售，以自主知识产权的核心技术为基础，为高端设备制造商提供个性化解决方案。公司作为全球领先的工业自动化产品和解决方案供应商，致力于以新质生产力突破中国并走向世界。

## 2. 飞书给客户带来的新能力和价值

### 2.1 具体使用的飞书功能模块

根据案例分析，汇川技术主要使用了以下飞书功能：

1. **飞书多维表格**: 核心应用工具，用于项目管理、预算管理和任务管理
2. **飞书文档**: 用于项目文档协作和信息共享
3. **飞书会议室**: 用于会议管理和协作
4. **飞书招聘**: 用于人力资源招聘管理
5. **飞书绩效**: 用于员工绩效管理

### 2.2 解决的业务痛点

#### 2.2.1 项目管理痛点
- **问题**: 缺乏多项目管控的数字化看板
- **具体表现**: 
  - 项目管理信息传递流程冗长、准确率低
  - 数据搜集阶段更新错误和数据丢失频繁
  - 数据分析操作繁琐，需要手动整理和同步
  - 信息更新复杂，发送异常通报需发送14封邮件
- **影响**: 增加时间、人力和沟通成本，导致项目管理不规范，交付效率低下

#### 2.2.2 预算管理痛点
- **问题**: 数据规模庞大、数据整合困难
- **具体表现**:
  - 涉及部门和人员超过100个，数据量巨大
  - 预算编制需要手动合并1000+表格
  - 预算模板分散，业务部门不熟悉，沟通不畅
  - 传统Excel工具因数据量过大出现卡顿和闪退
  - 数据整合方式纯手工，极易出现错漏
- **影响**: 耗费60%工作时间和精力，影响工作效率，带来财务安全风险

#### 2.2.3 任务管理痛点
- **问题**: 规范性和安全性风险
- **具体表现**:
  - 表格工具数据不完整、准确性差
  - 数据分散存放，填写格式不一致，更新不同步
  - 多软件项目管理难以追踪责任人和进展
  - 创建高级表格技术门槛高，步骤复杂
  - 每周花费4小时收集整合数据
- **影响**: 数据安全难以把控，时间和人力成本不必要增加
