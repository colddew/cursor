# 甄零科技 - 飞书客户案例分析报告

## 客户基本信息

**公司名称**：甄零科技
**所属行业**：企业服务（合同管理软件）
**公司规模**：未明确提及（从上下文推断为中小型创业公司）
**主营业务**：专注于"合同价值全生命周期管理"，致力于为客户打造业财法一体的内控体系，通过链接企业业务系统，由合同驱动执行系统运转，旨在为企业带来规范化、精细化的合同管理体验，同时加强企业风险防范能力。

## 飞书给客户的新能力/赋能

### 1. 会议效率革命性提升
- **飞书妙记自动会议记录**：彻底解决了重要会议靠人工记录的问题，会议内容可自动沉淀并一键分享给所有参会人
- **"飞阅会"流程优化**：开会前先在飞书文档中写讨论内容，会议中默读内容并写评论，结尾@对应人记下todo，整个会议流程比原先节省了一半时间
- **会议内容复用**：技术会议和客户会议的干货内容可用于公司培训，增强内容复用性

### 2. 项目管理全链路数字化
- **多维表格统一管理**：一张表轻松管理公司所有项目，解决了项目信息分散在各种文档和群聊中的混乱状态
- **项目全局可视化**：项目负责人可以清晰直观地查看公司整体项目数量、每个项目进展、延期情况以及人力分配
- **权限分级管控**：不同人员看到不同内容，公司管理者看到项目全局，项目执行者看到自己权限对应部分，保证信息安全

### 3. 组织协同效率大幅提升
- **"个人说明书"文化**：员工之间通过个人说明书更好了解彼此，降低协作成本；管理层将管理理念、沟通风格和业务思考放在文档中，降低管理成本
- **信息透明文化建设**：打造了信息透明的公司文化，提升内信息流动效率
- **跨部门协作优化**：打破了部门间的信息壁垒，实现了更高效的协同

### 4. 系统整合与All in One办公
- **开放平台集成能力**：飞书良好的开放性让甄零科技能够便捷地接入外部系统
- **多系统统一入口**：从需要登录5-6个账号（HR最多十几个账号）到只需登录飞书1个账号
- **应用敏捷开发**：支持集成CRM、OA、ERP、财务等多种场景，打破系统间壁垒
- **办公管理All in One**：实现了办公管理的All in one，减少了账号密码管理负担

## 飞书没有满足客户需求的部分

从案例内容来看，甄零科技对飞书的使用非常满意，没有明确提及未满足的需求或痛点。案例呈现的是一个相对完整和成功的数字化转型故事。

但从分析角度，可能存在的潜在需求包括：

1. **深度行业定制化**：作为专业的合同管理软件公司，可能需要更深入的行业特定功能集成
2. **高级数据分析能力**：虽然多维表格提供了项目管理功能，但可能还需要更强大的商业智能分析工具
3. **客户门户集成**：作为B2B服务公司，可能需要更好的客户协同和门户功能
4. **合规性特定需求**：合同管理涉及法律合规，可能需要更专业的合规性工具集成
