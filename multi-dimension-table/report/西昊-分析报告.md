# 西昊（SIHOO）飞书客户案例分析报告

## 1. 公司基本信息

**企业名称**：深圳市西昊智能家具有限公司
**成立时间**：2011年
**行业归属**：人体工学家具/智能家居行业
**主营业务**：人体工学椅、升降书桌及人体工学家具周边产品的研发、设计、制造、销售和服务
**企业规模**：近800名员工，拥有超10万平方米生产基地
**市场地位**：
- 连续2021-2024年京东、天猫电脑椅类目销售额第一
- 一年销量达150万件
- 产品畅销全球100多个国家和地区
- 获得国家专利130余项
- 深圳市人体工程学应用协会理事长单位，主导制定《人体工程学办公椅技术白皮书》等行业标准
- 国家级高新技术企业

## 2. 飞书应用场景

### 2.1 产品管理场景
- **多维表格应用**：搭建国内办公椅电商平台产品信息库，维护行业Top商品销售信息及相关数据
- **飞书词典应用**：建立包含所有产品关键信息的企业词典词条，包括单品图片、产品方案、核心卖点说明书等
- **云文档协同**：实现产品信息的集中管理和实时更新

### 2.2 物流仓储场景
- **RPA自动化**：多平台信息RPA抓取，机器人实时消息提醒
- **物流配货自动化**：通过飞书开放平台实现电商物流配货流程自动化

### 2.3 客服场景
- **多维表格客诉管理**：一张表集合全平台客诉信息，多维度展示客诉情况、原因追踪、跟进人员、反馈渠道
- **飞书机器人**：实现实时消息提醒和消息触达一体化
- **仪表盘看板**：通过对应仪表盘看板快速查阅问题数量和分布情况

## 3. 新获得的能力

### 3.1 数据管理与分析能力
- **实时数据监控**：通过多维表格实现市场动态实时监控，解决原始系统数据筛选不够灵活、数据过期问题
- **产品信息集中化**：建立统一的产品信息库，解决产品档案信息散乱问题
- **客诉数据可视化**：通过仪表盘看板和画册视图实现客诉数据的直观展示和分析

### 3.2 流程自动化能力
- **RPA自动化流程**：实现多平台信息自动抓取和机器人实时消息提醒
- **物流配货自动化**：电商物流配货流程实现自动化处理
- **客服流程标准化**：电商平台客诉日清制度，提高客服响应效率

### 3.3 协同办公能力
- **跨部门协同**：打破信息孤岛，实现销售、运营、产品等部门的信息同步
- **知识管理**：通过飞书词典建立企业知识库，实现产品信息的快速查询和共享
- **移动办公**：员工可以随时随地查看到需要的产品信息

### 3.4 客户服务能力
- **一站式沟通**：打通电商客服系统，实现消息触达一体化
- **客诉追踪与分析**：实时跟踪客诉处理情况，便于及时干预和优化
- **服务响应速度提升**：以最快速度响应用户需求和问题

## 4. 未满足的需求

### 4.1 系统集成需求
- **外部客服系统割裂**：应用独立的外部客服系统，需要从飞书跳出或通过人工反馈方式分发信息
- **多平台数据同步**：仍需通过RPA抓取多平台信息，未实现系统间的原生集成

### 4.2 数据处理能力
- **图文信息二次处理**：大量的图文信息仍需二次处理，导致客服频繁来回沟通
- **Excel依赖**：客服团队原本使用Excel进行数据汇总，虽然通过飞书改善，但仍需处理历史数据格式

### 4.3 业务流程优化
- **人工依赖**：基础而重复的问题仍需要人工回复，大量人力耗费在回复产品问题上
- **跨部门协作效率**：部门间配合与信息同步仍需依赖人的主动交流，跨部门协同效率有提升空间

### 4.4 知识管理深度
- **知识沉淀空间**：知识沉淀与知识获取仍有优化空间
- **产品信息查询**：随着产品不断更新迭代，不同产品线信息量增大带来的查询和管理困难

- **客诉数据可视化**：通过仪表盘看板实现客诉情况的直观展示和分析

- **制造企业**：生产、销售、客服等环节的协同需要高效的数字化工具支撑
