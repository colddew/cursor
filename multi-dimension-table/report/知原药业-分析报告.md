# 知原药业飞书客户案例分析报告

**核心业务：**

- 江苏省工程技术研究中心

- ISO14001环境管理体系认证
- ISO45001职业健康安全管理体系认证

## 2. 飞书应用场景

### 2.1 核心应用场景
知原药业通过飞书构建了统一数字化协同办公平台，主要应用于以下三大核心场景：

#### 场景一：多系统集成的一站式协同办公平台
- **应用功能：** 飞书集成平台、即时消息
- **解决方案：** 将分散的多套业务管理系统集成，打造统一入口
- **具体实现：**
  - OA平台统一处理工单、审阅合同并集中归档
  - 商业拓展团队集中管理对外联系人和客户生命周期
  - 管理层通过统一报表平台查阅各部门业务进展
  - 建立基于飞书的消息中心和审批中心

#### 场景二：基于多维表格的任务管理系统
- **应用功能：** 飞书多维表格
- **解决方案：** 搭建热舞管理，实现任务自动分配、多人协同、看板自动生成
- **具体实现：**
  - 自动化任务通知对应责任人
  - 多人在线实时跟进进展
  - 自动生成最新任务看板并发送给相关领导

#### 场景三：高效视频会议系统
- **应用功能：** 飞书视频会议、飞书文档、飞书妙记
- **解决方案：** 打破时空限制、信息同步升级、流程标准化沉淀
- **具体实现：**
  - 多端接入与实时屏幕共享
  - 语音转文字、重点内容标记及待办
  - 自动生成带时间戳的会议记录

### 2.2 技术集成情况
- **自建应用数量：** 50+
- **系统集成数量：** 15大系统
- **核心业务系统：** 基于飞书多维表格搭建

## 3. 新获得的能力

### 3.1 数字化整合能力
- **业务中台化：** 实现了从分散系统到统一平台的转变，消除"系统烟囱"
- **消息审批集中化：** 所有系统消息和审批流程统一到飞书平台，避免处理延时或遗漏
- **数据资产集中存储：** 解决了组织数据资产分散存储的问题

### 3.2 协作效率提升
- **跨部门协作效率：** 大幅降低跨部门沟通成本，解决责任不明确、信息传递不畅等问题
- **任务管理自动化：** 实现任务自动分配、实时更新、自动看板生成
- **会议质量提升：** 会议内容高效留存和便捷分享，员工体验显著改善

### 3.3 业务流程优化
- **工单处理流程：** 统一入口处理工单、审阅合同并集中归档
- **客户管理流程：** 集中管理对外联系人和客户整个生命周期
- **审批流程优化：** 基于飞书构建统一审批中心

### 3.4 量化成果
- **效率提升：** 基于飞书多维表格搭建的核心业务系统，效率提升30%
- **会议质量：** 会议治理效益明显，会议质量提高50%
- **系统集成：** 搭建50+自建应用，集成对接15大系统

## 4. 未满足的需求

### 4.1 系统整合挑战
- **系统割裂问题：** 在引入飞书前，各业务系统之间存在明显的"烟筒效应"，导致数据孤岛现状
- **重复性工作：** 由于系统割裂，一件小事需要来回咨询多人才能解决，沟通成本居高不下
- **数据完整性与有效性挑战：** 在外部审计时，基于邮件、excel、微信等工具开展的业务面临数据完整性和有效性挑战

### 4.2 任务管理痛点
- **任务信息滞后：** 任务进度、任务风险、任务统计等信息无法实时同步
- **通知跟进工作量大：** 需要通过邮件、IM工具手动发送任务给负责人，耗时长、任务重
- **协同割裂：** 任务跟进人无法同时协作，需要人工整合完成情况
- **进度查看滞后：** 由于数据传递链长，任务看板更新滞后，无法及时规避任务延期风险

### 4.3 会议管理问题
- **会议质量低下：** 会议数量多但质量不高，存在开会迟到、会议待办无法有效追踪等问题
- **会前协同成本高：** 需要逐一通知参与人，沟通确认参会事项
- **会议内容难以沉淀：** 会议入口不统一，会议内容难以留存和分享

### 4.4 合规风险
- **审计数据缺失：** 线下处理方式无法留下系统的审计数据，带来合规风险
- **组织数据资产损失：** 数据无法集中存储造成的损失
