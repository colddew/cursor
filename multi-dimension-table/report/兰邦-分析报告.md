# 兰邦飞书客户案例分析报告

## 1. 客户基本信息

### 公司概况
- **公司名称**：兰邦
- **所属行业**：制造业（特种纤维制品）
- **公司规模**：中型制造企业（拥有4个厂区）
- **成立时间**：2005年
- **主营业务**：主营特种纤维制品的科技型企业，致力于结合高性能新材料的独有特性，开发定制化的功能性产品，为安全、防护、工程等多个领域提供富有价值的解决方案

### 业务特点
- 产品远销海内外，具有国际化业务特征
- 处于快速上升期，业务需求和组织结构日趋复杂
- 典型的制造业企业，注重生产计划和供应链管理

## 2. 飞书带来的新能力和价值

### 2.1 使用的飞书功能模块

1. **飞书多维表格**：核心应用，用于搭建销售管理系统和采购管理系统
2. **飞书招聘**：用于规范招聘流程，提高面试效率
3. **飞书绩效**：与招聘配合，实现人才管理闭环
4. **飞书Office**：优化组织内信息化建设和协同效率
5. **飞书People**：实现了从招聘、管理、考核到晋升的完整闭环
6. **飞书会议室**：支持面试等场景

### 2.2 解决的业务痛点

#### 销售管理痛点
- **客户资源分散**：客户资源分布在不同销售手上，没有统一客户池
- **跟进记录分散**：销售跟进记录分散在Word文档里，难以追踪客户进展
- **销售预测困难**：依赖Excel做销售预测，难以查找和比对数据变化

#### 招聘管理痛点
- **流程不规范**：招聘流程缺乏规范，单次面试耗时长
- **信息无留痕**：面试过程无留痕，难以回顾面试内容
- **招聘质量不稳定**："只要老板不面试，招聘质量就下降"，业务部门面试能力不足

#### 采购与生产协同痛点
- **沟通方式落后**：依赖纸质单和微信沟通，单据混乱
- **信息不同步**：采购与生产部门信息不同步，协同性差
- **物料管理困难**：物料领取情况无法及时准确掌握

### 2.3 带来的业务价值和效率提升

#### 销售管理价值
- **系统化管理**：搭建一站式销售管理系统，让销售过程管理更透明
- **结构化数据**：建立结构化的客户数据管理，按客户维度拆分
- **实时监控**：实时数据看板，掌握销售额变动和客户分布情况
- **成本节约**：**节省20万CRM搭建成本**

#### 招聘管理价值
- **流程标准化**：规范招聘流程，确保每位候选人经过两轮业务面试
- **过程留痕**：面试过程全程留痕，便于回顾和评估
- **效率提升**：面试时间从2小时缩短至40分钟
- **规模化招聘**：**面试量提升至原来的3倍**，有机会接触更优秀候选人
- **能力提升**：面试界面集成简历信息、过往评价和题库，提升面试能力

#### 采购管理价值
- **流程自动化**：采购流程线上化，自动化推进采购管理节点
- **信息同步**：物料领用进度自动更新，信息同步及时
- **效率提升**：**领料时间节省30%**
- **规模化推广**：覆盖4个厂区，推动数字化流程普及
