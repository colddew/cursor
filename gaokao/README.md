# 高考院校数据项目

基于阳光高考网和教育部官方名单的全国普通高等学校数据整理项目。

## 数据概览

- **总学校数**: 2915所
- **数据完整性**: 100%（学校名称、省份、城市、主管部门、办学层次、院校特性）
- **更新时间**: 2026-01-11

## 项目文件

### 数据文件

| 文件 | 说明 |
|------|------|
| [招生章程.xlsx](招生章程.xlsx) | 主数据文件，包含2915所学校的完整信息 |
| [2025全国普通高等学校名单.xlsx](2025全国普通高等学校名单.xlsx) | 教育部官方名单，作为数据源参考 |

### 脚本文件

| 文件 | 说明 |
|------|------|
| [crawl_gaokao_schools.py](crawl_gaokao_schools.py) | 阳光高考网爬虫，获取学校基础信息 |
| [fill_features_by_tags.py](fill_features_by_tags.py) | 基于筛选标签补充院校特性数据 |
| [fill_city_data.py](fill_city_data.py) | 从教育部名单匹配城市数据 |
| [fill_47_cities.py](fill_47_cities.py) | 补充47所匹配失败院校的城市信息 |
| [add_remarks.py](add_remarks.py) | 添加备注列，标注匹配失败原因 |

## 城市数据填充方式

1. **阳光高考网原始数据**（216所，7.4%）
   - 直辖市学校自带城市信息

2. **教育部官方名单匹配**（2652所，91.0%）
   - 标准化学校名称后精确匹配
   - 解决全半角括号等问题

3. **网络搜索补充**（47所，1.6%）
   - 16所军事院校
   - 13所双一流分校/医学院
   - 9所成人教育类院校
   - 9所其他特殊情况

## 使用说明

### 查看数据

使用 Excel 或 pandas 打开 `招生章程.xlsx`：

```python
import pandas as pd
df = pd.read_excel('招生章程.xlsx')
print(df.head())
```

### 重新运行脚本

```bash
# 补充院校特性（如需要重新爬取）
python3 fill_features_by_tags.py

# 填充城市数据
python3 fill_city_data.py

# 补充47所院校城市
python3 fill_47_cities.py

# 添加备注列
python3 add_remarks.py
```

## 注意事项

1. 院校特性列为空是正常的，表示该学校为公办高校
2. 47所院校的备注列说明了匹配失败的原因
3. 城市数据已100%填充，无需额外处理
