# 高考院校数据项目

基于阳光高考网和教育部官方名单的全国普通高等学校数据整理项目。

## 数据概览

- **总学校数**: 2915所
- **数据完整性**: 100%（学校名称、省份、城市、主管部门、办学层次、院校特性）
- **数据结构**: 15列（含拆分的招生章程详情页4列）
- **招生章程详情页链接**: 97.0%（2827所有效链接）
- **多链接学校**: 11所（已拆分为本部/特殊两列）
- **0-标记学校**: 88所（暂无2025年招生章程）
- **更新时间**: 2026-01-15

## 城市数据填充方式

1. **阳光高考网原始数据**（216所，7.4%）
   - 直辖市学校自带城市信息

2. **教育部官方名单匹配**（2652所，91.0%）
   - 标准化学校名称后精确匹配
   - 解决全半角括号等问题

3. **网络搜索补充**（47所，1.6%）
   - 16所军事院校
   - 13所双一流分校/医学院
   - 9所成人教育类院校
   - 9所其他特殊情况

## 使用说明

### 查看数据

使用 Excel 或 pandas 打开 `招生章程.xlsx`：

```python
import pandas as pd
df = pd.read_excel('招生章程.xlsx')
print(df.head())
```

### 验证数据准确性

使用本地 Playwright 脚本验证招生章程链接是否仍然有效：

```bash
# 直接运行（会从进度文件恢复）
python3 verify-with-playwright.py

# 重新开始（删除进度文件）
rm verification_progress.json
python3 verify-with-playwright.py
```

**特点**：
- 本地运行，不消耗 Token
- 20条/批，支持断点续传
- 自动创建验证副本：`招生章程_验证副本.xlsx`
- 反爬虫配置已验证可用

### 重新运行脚本

```bash
# 补充院校特性（如需要重新爬取）
python3 fill_features_by_tags.py

# 填充城市数据
python3 fill_city_data.py

# 补充47所院校城市
python3 fill_47_cities.py

# 添加备注列
python3 add_remarks.py
```

## 注意事项

1. 院校特性列为空是正常的，表示该学校为公办高校
2. 47所院校的备注列说明了匹配失败的原因
3. 城市数据已100%填充，无需额外处理
