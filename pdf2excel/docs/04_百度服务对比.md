# 百度 OCR 服务平台对比

## 🏢 两个不同的平台

### 1. 百度智能云（商业平台）

**定位**：商业化云服务
**网址**：https://ai.baidu.com/

**免费额度**（需实名认证）：
- 文档解析（PaddleOCR-VL）：**200页**（一次性）
- 通用文字识别：个人 1000次/月，企业 2000次/月

**付费方式**：
- 按量后付费：￥0.18/页
- 预付费资源包：有折扣

**认证方式**：
- API Key + Secret Key
- 需要创建应用

---

### 2. AI Studio（学习/开发平台）

**定位**：AI 学习和开发平台
**网址**：https://aistudio.baidu.com/paddleocr

**免费额度**：
- ⚠️ **没有明确的免费额度说明**
- 超额会返回 **429 错误**（Too Many Requests）
- 主要用于学习和测试

**认证方式**：
- 部署服务后获得专属 API URL
- 使用 Access Token

---

## 🤔 应该选哪个？

### 场景1：学习测试（少量）
**推荐**：AI Studio
- 免费
- 快速上手
- 适合测试

### 场景2：生产使用（大量）
**推荐**：百度智能云
- 明确的免费额度（200页）
- 稳定可靠
- 有技术支持
- 超出后可付费

---

## ⚠️ 重要区别

| 项目     | 百度智能云    | AI Studio       |
| -------- | ------------- | --------------- |
| 定位     | 商业服务      | 学习平台        |
| 免费额度 | 200页（明确） | 未明确（会429） |
| 稳定性   | 高            | 一般            |
| 技术支持 | 有            | 社区            |
| 适合场景 | 生产环境      | 学习测试        |
