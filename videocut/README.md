# 视频自动剪辑工具

这是一个基于 Python 的视频自动剪辑工具，主要用于处理口播视频的编辑需求。

## 需求说明

1. 静音处理
   - 自动检测视频中的静音片段（气口）
   - 将超过 500ms 的静音片段压缩为 500ms
   - 保留短于 500ms 的静音片段

2. 重复内容处理
   - 自动检测视频中连续的相似语音片段
   - 当说错话重复说时，只保留最后一次正确的表述
   - 基于音频特征分析，无需网络连接

3. 背景音乐
   - 支持添加背景音乐
   - 自动调整背景音乐长度匹配视频
   - 背景音乐音量自动调整为原音频的 30%

4. 字幕显示
   - 支持添加白色点宋体字幕
   - 字体大小为11号
   - 字幕位置在视频从下往上高度的1/3处
   - 字幕居中显示
